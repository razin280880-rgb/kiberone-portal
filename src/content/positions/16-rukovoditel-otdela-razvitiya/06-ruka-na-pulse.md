# РОР: рука на пульсе

Оперативный мониторинг и эскалация. Что РОР видит каждый день / неделю / месяц, какие сигналы — тревожные, кому эскалирует.

---

## Дневной мониторинг (8:00–17:00)

### Утро (8:15–8:45)

**Что смотрим:**
- Дашборд входной воронки за вчера: анкеты по городам, по каналам
- Чат с РОП: жалобы / похвала по качеству анкет
- Чат с маркетологом: что с платными каналами за ночь
- Очередь анкет «не переданных в РОП» — есть ли застрявшие > 24 часов
- Статус промо-точек: все ли промоутеры вышли (отчёт от менеджеров развития к 09:00)

### В течение дня

- Письменный фидбэк по 5–10 звонкам менеджеров развития
- Реакция на «горящие» сообщения от менеджеров развития (отказ школы, упавшая точка, конфликт с площадкой)
- Прогресс по запланированным синкам, 1-1, обучениям

## Тревожные сигналы (когда «загорается красная лампочка»)

### 🔴 Критические — реакция в течение часа

| Сигнал | Что означает | Действие |
|---|---|---|
| Поток анкет на день в городе **= 0** | Промоутеры не вышли / точка закрылась / менеджер не работает | Звонок менеджеру развития города, выяснение, восстановление потока к концу дня |
| Хантер в чате: «третья грязная анкета подряд от Х» | Системный провал квалификации | Звонок менеджеру, разбор, регламент на следующий день. При повторе через неделю — внеочередная аттестация |
| Менеджер развития не вышел на планёрку 08:00 без уведомления | Дисциплина / возможно увольнение / выгорание | Личный звонок в 08:30, выяснение причины, при отсутствии связи — эскалация в HR / РФ города |
| Жалоба от директора школы / партнёра | Удар по репутации канала | Перехватить лично в течение дня, провести разбор инцидента |
| Падение конверсии «анкета → запись на пробное» ниже 30% за неделю по городу | Скрипт сломан / менеджер «выгорел» / канал испортился | Прослушать 20+ звонков менеджера за неделю, найти причину, скорректировать |

### 🟡 Предупреждающие — реакция в течение 1–3 дней

| Сигнал | Что означает | Действие |
|---|---|---|
| План недели по анкетам в городе выполняется < 80% к четвергу | Риск не выйти на план месяца | Звонок менеджеру развития, пересмотр плана пятницы и субботы, возможно усиление промо |
| Доля «грязных» анкет в городе > 15% за неделю | Регламент квалификации нарушается | Полный разбор всех «грязных» анкет с менеджером + ролевая игра |
| Стоимость лида (CAC) по платному каналу выросла > 30% за неделю | Канал «остыл» / аукцион перегрет | Совместный разбор с маркетологом, корректировка / временная пауза |
| Промоутер на точке снизил эффективность в 2 раза | Выгорание / нашёл другую работу / точка остыла | Прослушать звонки промоутера, выйти на точку, при необходимости — замена |
| Менеджер развития не закрыл задачи прошлой недели | Проблема приоритизации / перегрузка | 1-1 в течение 2 дней, пересмотр загрузки |

### 🟢 Внимательные — фиксируем, наблюдаем

| Сигнал | Что означает | Действие |
|---|---|---|
| Появление нового конкурента в одном из городов | Возможен отток лидов / повышение CAC | Анализ как меняется качество и стоимость лидов в этом городе; через 30 дней — решение |
| Сезонный спад (декабрь, февраль) | Нормальная сезонность | Заранее перестроить каналы (больше школ / меньше промоутеров на улице зимой) |
| Менеджер развития сменил тон в общении | Возможное выгорание / поиск работы | Личное 1-1 «на чай», узнать ситуацию |

## Недельный мониторинг (пятница 17:00–18:00)

**Что смотрим:**
- Дашборд за неделю: план/факт по всем городам и каналам
- Все «протухшие» анкеты (> 7 дней без касания) — почему, чья ответственность
- Все «грязные» анкеты недели — разбор каждой
- Конверсия квалификации недели — где провал, где рост
- Стоимость лида (CAC) за неделю — есть ли каналы с резким изменением
- Результаты обучающей сессии: что разобрали, что в работу
- Статус активных партнёрских договорённостей — что продвинулось, что застряло

**Артефакт недели:** короткий отчёт (3–5 пунктов) для собственника к понедельнику.

## Месячный мониторинг (первые 3 рабочих дня месяца)

**Что смотрим:**
- Дашборд за месяц: план/факт целиком
- Поканальный анализ: ROI, CAC, конверсия каждого канала
- Топ-3 «удачи» месяца (что сработало неожиданно хорошо) + топ-3 «провала» (что не получилось)
- Аттестация каналов: какие масштабируем, какие закрываем, какие тестируем
- Аттестация менеджеров развития (если месяц замыкает полугодие)
- Готовится квартальный план (если месяц замыкает квартал)

**Артефакт месяца:** ежемесячный отчёт для собственника + презентация на стратегической планёрке первого понедельника нового месяца.

## Эскалация — кому и когда

| Ситуация | Эскалация |
|---|---|
| Провал плана по анкетам в городе 2 недели подряд | РОР → собственник на стратегической планёрке |
| Системный провал по «грязным» анкетам у одного менеджера | РОР → внеочередная аттестация (с уведомлением собственника) |
| Утрата ключевого партнёрства (крупная школа, важная площадка) | РОР → собственник немедленно |
| Жалоба от партнёра / директора школы / РФ на менеджера развития | РОР разбирает + уведомляет собственника / ОД |
| Сбой по платному каналу (резкий рост CAC, падение качества) | РОР + маркетолог → собственник на ближайшем синке |
| Конкурент агрессивно работает в городе (демпинг, переманивание тьюторов) | РОР → собственник + РФ города → коллективное решение |
| Кадровая проблема (нужен новый менеджер / увольнение слабого) | РОР → HR + собственник |

| С кем | Когда | Зачем |
|---|---|---|
| Собственник / ОД | Понедельник 10:00 (стратегическая планёрка) + 1-1 по запросу | Стратегические решения, эскалации |
| РОП | Пятница (1 час) | Качество анкет, передача в продажу |
| Маркетолог | Пятница (30–45 мин) | Платные каналы, креативы, бюджет |

| HR | По запросу + раз в месяц | Найм / адаптация / увольнения |
| Методист | По запросу | Школьные мероприятия, мастер-классы |

## Журналы РОР (обязательные документы)

1. **Журнал планёрок** (ежедневная + еженедельная разборная)
2. **Журнал прослушиваний** — кого слушал, какой балл, что сказал
3. **Журнал партнёрств** — все школы и партнёры со статусом
4. **Журнал промо-точек** — все точки с состоянием
5. **Журнал каналов** — тесты, решения, ROI
6. **Журнал аттестаций отдела** (раз в полугодие на каждого сотрудника)
7. **Журнал обучающих сессий** (что разбирали, материалы)
8. **Журнал инцидентов** — провалы, жалобы, эскалации (для аналитики паттернов)

Все журналы хранятся в общедоступном пространстве (Google Drive / Notion), доступны собственнику / ОД.

## Когда РОР сам становится «красной лампочкой» для собственника

Если у самого РОР начинаются паттерны:

- 🚩 Пропускает планёрки 08:00 → собственник сразу обсуждает причину
- 🚩 Не выезжал в города 2+ месяца → теряет руку на пульсе
- 🚩 Дашборд не обновляется ≥ 5 дней → потеря оперативной картины
- 🚩 Менеджеры развития не имеют свежих 1-1 (≥ 1 мес) → ритмика управления провалена
- 🚩 Доля «грязных» анкет в сети > 15% месяц подряд → системный провал отдела
- 🚩 «Сам поехал в школу / сам делает квалификацию» больше 30% времени → играющий тренер

В этом случае — **внеочередная аттестация РОР собственником** + план развития / решение о замене.
