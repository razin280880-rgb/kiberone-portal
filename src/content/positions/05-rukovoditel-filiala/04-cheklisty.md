# Чек-листы руководителя филиала

Все чек-листы — рабочий инструмент. Заполнение фиксируется (галочка с датой), хранится минимум 12 месяцев.

**Логика:** РФ управляет 10-20 людьми, координирует 5 центральных функций, контролирует P&L. Без чек-листов он либо тонет в оперативке, либо теряет ключевые управленческие задачи.

---

## ЧЕК-ЛИСТ 1. Ежедневный (заполняет в конце дня)

**Дата:** _______ **РФ:** _______ **Город:** _______

### Управление командой
- [ ] Утренняя планёрка проведена (15 минут)
- [ ] Все плановые 1-1 на сегодня — состоялись
- [ ] Чат с командой проверен 2 раза за день, ключевые вопросы — отвечены

### Операционка
- [ ] Дашборд филиала проверен утром (план/факт по KPI)
- [ ] Тревожные сигналы за вчера обработаны или в работе
- [ ] Реактивные эскалации (конфликты, жалобы) — есть в работе или закрыты

### Связь с центром
- [ ] Если был синк с центральной функцией — есть конкретные договорённости
- [ ] Эскалации к собственнику (если нужны) — отправлены

### Полевая работа
- [ ] Минимум 1 «полевое» событие сегодня (урок / клиент / школа), кроме офиса

### Финансы
- [ ] Если были запросы на расходы — приняты решения

### Документы
- [ ] Заметки за день внесены в дневник РФ
- [ ] План на завтра составлен

**Если хотя бы одна галочка не стоит** — день не закрыт. На следующий день — приоритет догнать.

---

## ЧЕК-ЛИСТ 2. Недельный (заполняет в пятницу)

**Неделя с _____ по _____** **РФ:** _______

### Управление командой
- [ ] Минимум 5 1-1 встреч с командой за неделю
- [ ] Понедельничная общая планёрка проведена
- [ ] Пятничный итог недели проведён
- [ ] Новички / стажёры — пообщался с каждым

### Операционные KPI (за неделю)
- [ ] Лидген — план/факт зафиксированы
- [ ] Продажи — план/факт зафиксированы
- [ ] Удержание — статус известен
- [ ] Качество (по методисту) — статус известен

### Связь с центром
- [ ] Синк с собственником прошёл (минимум 1 раз за неделю)
- [ ] Минимум 2 синка с центральными функциями

### Стратегия
- [ ] Если были стратегические задачи на неделю — есть прогресс
- [ ] Если есть открытые «быстрые победы» — есть статус

### Тревожные сигналы (см. `06-ruka-na-pulse.md`)
- [ ] Все открытые сигналы > 7 дней — есть план или закрыты

### Документы
- [ ] Дашборд недели заполнен
- [ ] Отчёт за неделю отправлен собственнику
- [ ] Заметки за неделю систематизированы

### Следующая неделя
- [ ] План на неделю составлен и согласован
- [ ] Все встречи в календаре

## ЧЕК-ЛИСТ 3. Ежемесячный (последняя пятница)

**Месяц _____** **РФ:** _______ **Город:** _______

### Финансы (главный блок)
- [ ] P&L филиала за месяц закрыт
- [ ] Выручка vs план: ___ %
- [ ] Расходы vs бюджет: ___ %
- [ ] Прибыль vs план: ___ %
- [ ] Если отклонения > 10% — анализ причин и план корректировки

### Команда
- [ ] Все ключевые сотрудники получили 1-1 в этом месяце (минимум 1 раз)
- [ ] Опрос команды (если по графику) — проведён, результаты обработаны
- [ ] Текучка за месяц: ___ человек ушло, ___ нанято
- [ ] Тьюторы в зоне риска (по фидбэку методиста) — есть план

### Операционка
- [ ] Лидген: ___ анкет (норма ___) — менеджер развития получил обратную связь
- [ ] Продажи: конверсия ___% (норма ___) — РОП получил обратную связь
- [ ] Удержание: отток ___% (норма ≤ 8%) — фермер получил обратную связь
- [ ] Качество: средний балл ___ (норма ≥ 7.5) — методист получил обратную связь

### Стратегия
- [ ] Минимум 1 стратегическая инициатива за месяц — есть прогресс
- [ ] Презентация результатов собственнику подготовлена
- [ ] Встреча с собственником прошла
- [ ] План на следующий месяц согласован

### Самооценка
- [ ] Заполнил мини-опрос для себя: что получилось, что — нет

## ЧЕК-ЛИСТ 4. Утренняя планёрка с командой (15 минут)

**Дата:** _____ **Участники:** _____

### Структура планёрки
- [ ] Начало вовремя (не позже 9:05)
- [ ] Каждый сотрудник озвучил план на день (по 1-2 минуте)
- [ ] Если есть «вопрос дня» — обсудили (5 минут)
- [ ] Если есть благодарности / поздравления — озвучили
- [ ] Завершение вовремя (≤ 15 минут общая длительность)

### Не делать на планёрке
- [ ] НЕ разбираю конфликты прилюдно (это в 1-1)
- [ ] НЕ делаю длинные монологи (мне 1-2 минуты максимум)
- [ ] НЕ обсуждаю слабых сотрудников при всех

### После
- [ ] Если кто-то просил отдельной встречи — поставил в календарь

## ЧЕК-ЛИСТ 5. Еженедельная планёрка с командой (30-60 минут, понедельник)

**Дата:** _____ **Участники:** _____

### Подготовка (за 1 час до)
- [ ] Сводка результатов прошлой недели готова (план/факт)
- [ ] Приоритеты на эту неделю сформулированы
- [ ] Презентация (если нужно) — подготовлена

### Структура планёрки
- [ ] Начало вовремя
- [ ] Краткий обзор результатов прошлой недели (10-15 мин)
- [ ] Озвучивание приоритетов на эту неделю по командам (15-20 мин)
- [ ] Каждая команда — короткий план (5-10 мин в сумме)
- [ ] Открытые вопросы / обмен (10-15 мин)
- [ ] Благодарности / поздравления (если есть)
- [ ] Завершение вовремя

### После
- [ ] Протокол планёрки сохранён в общем чате
- [ ] Все договорённости — поставлены в трекер задач

## ЧЕК-ЛИСТ 6. 1-1 с сотрудником (раз в месяц с каждым ключевым)

**Дата:** _____ **С кем:** _____ **Длительность:** 30-45 минут

### Подготовка (за 10-15 минут)
- [ ] Прочитал актуальный статус сотрудника (KPI, последние результаты)
- [ ] Подумал, что обсудить (минимум 2-3 темы)
- [ ] Готов слушать, не «руководить»

### Структура встречи

**1. Как дела (5-10 минут)** — открытый вопрос
- [ ] Спросил, как сотрудник себя чувствует в работе
- [ ] Слушал, не перебивал

**2. Текущие задачи и блокеры (10-15 минут)**
- [ ] Что в работе сейчас?
- [ ] Что мешает / что нужно от меня?
- [ ] Если есть запросы — записал, буду решать

**3. Развитие и рост (5-10 минут)**
- [ ] Чему хочет научиться?
- [ ] Какие следующие шаги в карьере?
- [ ] Что я как РФ могу сделать для этого?

**4. Обратная связь (5-10 минут) — двусторонняя**
- [ ] Дал конкретную обратную связь по работе (что хорошо, что улучшить)
- [ ] Спросил обратную связь о себе как о руководителе

**5. Договорённости (5 минут)**
- [ ] Конкретные действия с обеих сторон, с датами

### После встречи
- [ ] Записал в дневник РФ: ключевые моменты, договорённости
- [ ] Если есть запросы от сотрудника — поставил себе задачу
- [ ] Через неделю — короткий чек-ин по результатам

## ЧЕК-ЛИСТ 7. Посещение урока (раз в неделю минимум 1 урок)

**Дата:** _____ **Тьютор:** _____ **Группа:** _____

### Что замечает РФ (отличается от методиста!)
**РФ смотрит не на учебное качество (это методист), а на:**

- [ ] Организация урока: вовремя начался, оборудование готово, материалы под рукой
- [ ] Атмосфера: дети вовлечены, не страшно, не скучно (общая оценка)
- [ ] Тьютор: уверен, спокоен, не «выгорел»
- [ ] Ассистент: реально работает или «занимает место»
- [ ] Помещение: чисто, технически ок
- [ ] Родители: впечатления (если есть, например, открытое занятие)

### После урока
- [ ] Кратко поблагодарил тьютора (не «разбор по качеству» — это методист)
- [ ] Записал заметки для себя
- [ ] Если что-то критическое (например, оборудование сломано) — оперативное решение
- [ ] Если есть фидбэк по тьютору как сотруднику (не как педагогу) — добавлю в его 1-1

## ЧЕК-ЛИСТ 8. Месячный синк с собственником (1 час)

**Дата:** _____ **Месяц:** _____

### Подготовка (за 24 часа до)
- [ ] Презентация готова (1-2 страницы): результаты месяца, вызовы, план следующего
- [ ] Все KPI зафиксированы
- [ ] Список «нужно решений от собственника» — составлен (3-5 пунктов)

### Структура встречи

**1. Результаты месяца (15 мин)**
- [ ] P&L филиала: выручка, расходы, прибыль
- [ ] Операционные KPI: лидген, продажи, удержание, качество
- [ ] Команда: текучка, найм, проблемы

**2. Главное за месяц (10 мин)**
- [ ] Что хорошо
- [ ] Что плохо
- [ ] Что неожиданно

**3. Открытые вопросы / эскалации (15 мин)**
- [ ] Что нужно решить совместно с собственником
- [ ] Что нужно изменить в политике / стандартах сети

**4. Стратегия (10 мин)**
- [ ] Прогресс по стратегическим инициативам
- [ ] Гипотезы / предложения

**5. План следующего месяца (10 мин)**
- [ ] Согласование приоритетов
- [ ] Согласование KPI

### После встречи
- [ ] Протокол встречи отправлен собственнику в течение 24 часов
- [ ] Все договорённости — поставлены в работу

## ЧЕК-ЛИСТ 9. Найм нового сотрудника

**Должность:** _____ **Кандидат:** _____ **Дата:** _____

### Подготовка
- [ ] Согласована вакансия с собственником (если бюджетное решение)
- [ ] Описание вакансии готово (соответствует [`01-rol-i-funkcii.md`](01-rol-i-funkcii.md) для роли)
- [ ] HR подключён к процессу

### Собеседование
- [ ] Прочитал резюме до встречи
- [ ] Подготовил 5-7 вопросов под роль
- [ ] Длительность не менее 45 минут
- [ ] Ответил на вопросы кандидата

### Решение
- [ ] Согласовано с центральной функцией (методист — для тьюторов, маркетолог — для менеджера развития, etc.)
- [ ] Согласовано с собственником (если нужно)
- [ ] Оффер сделан в срок (обычно 1-2 дня после интервью)

### После найма
- [ ] Запущена программа онбординга (см. соответствующую папку для роли)
- [ ] Назначен наставник
- [ ] Все доступы / документы готовы к первому дню

## ЧЕК-ЛИСТ 10. Увольнение сотрудника

**Сотрудник:** _____ **Причина:** _____ **Дата:** _____

### Подготовка
- [ ] Согласовано с собственником (для ключевых ролей) и HR
- [ ] Документы (трудовое расторжение) готовы
- [ ] План передачи дел готов (кто берёт работу, на какой срок)

### Сама встреча с увольняемым
- [ ] Встреча 1-1, не публично
- [ ] Прямое сообщение: причина, решение, не «обвинение»
- [ ] Время на ответ: дать сотруднику возможность сказать
- [ ] Условия выхода: дата последнего дня, выплаты, документы

### После
- [ ] Команда информирована корректно (без деталей причины, если личное)
- [ ] Hand-over проведён
- [ ] Exit-интервью с HR проведено
- [ ] CRM / системы / доступы — закрыты в день ухода

## ЧЕК-ЛИСТ 11. Запуск нового продукта в городе (например, летний лагерь)

**Продукт:** _____ **Дата запуска:** _____

### За 6-8 недель до запуска
- [ ] Согласована программа с методистом (центральным)
- [ ] Согласован бюджет с собственником
- [ ] Команда (тьюторы, ассистенты) — назначены
- [ ] Промо-материалы готовы (с маркетологом)

### За 2-4 недели до запуска
- [ ] Активная продажа (через фермера + хантера)
- [ ] Контроль набора групп (минимум 50% — должен быть набран)
- [ ] Если набор слабый — корректировки (продлить продажи / усилить рекламу)

### За 1 неделю до запуска
- [ ] Финальная проверка: команда готова, оборудование, материалы
- [ ] Презентация для родителей (если предусмотрена) — готова

### После запуска
- [ ] Ежедневный мониторинг первых дней
- [ ] Сбор обратной связи родителей и детей
- [ ] Корректировки на ходу (если нужны)

### Постмортем (через 2 недели после запуска)
- [ ] Финансовый итог: выручка, расходы, прибыль
- [ ] Качественный итог: отзывы, что улучшить
- [ ] Документ «уроки на следующий запуск» (1 страница)

## КАК ХРАНЯТСЯ ЧЕК-ЛИСТЫ

**Где:**
- Google Drive: `KIBERone Management / Чек-листы РФ / [город] / [год] / [месяц] /`
- Каждый чек-лист — отдельный файл с датой
- Раз в квартал — архивируется

**Доступ:**
- РФ — пишет
- Собственник — читает
- HR — читает (для аттестаций)
- Методист — читает чек-листы посещений уроков

**Срок хранения:**
- Ежедневные / недельные — 12 месяцев
- Месячные — 24 месяца
- Найм / увольнение — бессрочно
- 1-1 встречи — 12 месяцев

**Правило:** если чек-лист не заполнен или потерян, считается, что соответствующая работа НЕ была сделана.
