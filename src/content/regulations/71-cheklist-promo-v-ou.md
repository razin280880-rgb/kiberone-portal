# Чек-лист подготовки и проведения промо в ОУ

**Кому:** менеджер развития, тьютор / спикер на МК, помощник на родсобрании.
**Формат:** компактный практический чек-лист на 1 печатную страницу — распечатать и держать в папке для каждого визита в школу.
**Расширенные регламенты:** [34-provedenie-master-class-v-shkole.md](34-provedenie-master-class-v-shkole.md), [64-script-roditelskoye-sobranie.md](64-script-roditelskoye-sobranie.md), [70-rukovodstvo-vzaimodejstviya-s-ou.md](70-rukovodstvo-vzaimodejstviya-s-ou.md).

---

## 🎯 Правило одной строки

**Не выходим в школу, если хотя бы один пункт чек-листа не закрыт.** Лучше перенести на день, чем провалить визит.

---

## 📅 За 1 неделю до визита

### Согласование со школой
- [ ] Дата и время **подтверждены** в письменном виде (WhatsApp / email)
- [ ] Уточнено: **сколько минут** даёт школа (45 — для МК, 10–15 — для родсобрания)
- [ ] Уточнено: **сколько детей / родителей** ожидается
- [ ] Уточнено: **есть ли проектор**, кабель HDMI
- [ ] Есть **контакт классрука** (телефон / WhatsApp) — он наш союзник на месте

### Согласие с родителями (только для МК)
- [ ] Классрук **проинформировал родителей** заранее о тематическом уроке (без этого риск жалобы)
- [ ] Есть **готовый текст** для родчата (из [33-soobshenie-roditelyam-posle-master-class.md](33-soobshenie-roditelyam-posle-master-class.md))

### Документы
- [ ] **Копия лицензии** в папке
- [ ] **Согласование с УО** (если требовалось) — в папке
- [ ] **Реквизиты юр.лица** — на случай вопросов директора

---

## 🎒 За 1 день — собираем пакет

### Анкеты и сбор
- [ ] **Бумажные анкеты** в количестве: `число родителей × 1.3` (для родсобрания) или `число детей в классе + 20%` (для МК → детям отдаём сертификаты с QR)
- [ ] **Ручки** — по 1 шт. на каждого, не одна на всех
- [ ] **Клипборд / папка** для собранных анкет
- [ ] **QR-код на анкету** — крупно на А4 (резерв, если бумажные не приняли)
- [ ] Текст согласия ПДн **на каждой анкете** (см. [67-script-personalnye-dannye.md](67-script-personalnye-dannye.md))

### Раздаточный материал
- [ ] **Сертификаты на бесплатное пробное** — по числу детей в классе + 20% запас
- [ ] **Брендированные наклейки / магниты** для детей (1 на ребёнка)
- [ ] **Флаеры с программой** — для тех родителей, кто захочет «подробнее»

### Контент
- [ ] **Презентация** загружена на флешку **+** есть резерв в облаке (Я.Диск / Google Drive)
- [ ] Презентация **открывается** — проверить на любом ноутбуке заранее
- [ ] Есть **резерв без проектора** — план выступления, если проектор сломался

### Реквизит (для МК с детьми — обязательно)
- [ ] **Меч из Minecraft / пиксельные очки / светящееся зелье** (что-то одно минимум)
- [ ] **Брендированный баннер** «Спасибо! Вы теперь цифровые супергерои!» — для фото
- [ ] **Наушники** (3–5 шт.) — как подарок самым активным детям

### Внешний вид
- [ ] **Деловой кэжуал** (не футболка, не костюм-тройка)
- [ ] **Бэйдж** с именем и должностью «KIBERone — школа цифрового творчества»
- [ ] Никакого запаха табака, парфюма — нейтрально
- [ ] Зарядка телефона — на 100%

### Для родсобрания дополнительно
- [ ] **3–5 фото с МК** (если был) — на телефоне или на флешке
- [ ] **Благодарственное письмо** для школы (1 шт., передать в конце)

---

## 🏫 В день визита — приехали в школу

### За 15 минут до начала
- [ ] **Пришли заранее** (не «впритык» — это нервы и провал)
- [ ] **Зашли через директорскую** — представились, поздоровались (даже если уже знакомы)
- [ ] **Установили проектор** / проверили презентацию
- [ ] **Разложили реквизит** в классе (но не отвлекающе — за столом учителя)
- [ ] **Спросили классрука**: «Есть ли особенности — гиперактивные дети, кто-то с особенностями развития, на кого опереться»

### Перед началом
- [ ] **Телефон на беззвучный**, ноутбук с уведомлениями выключенными
- [ ] **Бутылка воды** на столе (45 минут говорить)
- [ ] **Лицензия** на видном месте (для родсобрания) или в папке (для МК)

---

## 🎤 Во время выступления

### МК для детей
- [ ] **Менять формат каждые 5–7 минут** (рассказ → вопрос → история → активность)
- [ ] **Запоминать имена** 2–3 активных детей с самого начала, обращаться по имени
- [ ] **Не унижать** тех, кто отвлекается, **не спорить** с провокациями
- [ ] **Делать фото** (3–5 за урок) — с разрешения классрука
- [ ] **Закончить вовремя** — даже если идёт хорошо. 45 минут максимум

### Родсобрание
- [ ] **Уложиться в время** (10–15 минут, не больше)
- [ ] **Поднять руку в зале** минимум 1 раз — якорь внимания
- [ ] **Не называть конкретную цену** залу — только индивидуально
- [ ] **Раздать анкеты только после рассказа**, не до
- [ ] **Возражения отрабатывать коротко** — детально потом по телефону

---

## 📦 После выступления (сразу, не уходя)

### Сбор анкет
- [ ] **Собрали все анкеты** в одну стопку (классрук помогает)
- [ ] **Проверили галочку согласия ПДн** на каждой — без неё анкета невалидна
- [ ] **Зафиксировали число анкет** — сразу при приёме, при свидетелях

### Передача школе
- [ ] **Передали классруку текст** для родительского чата (если был МК)
- [ ] **Передали директору / завучу** благодарственное письмо
- [ ] **Сделали фото** с классруком / завучем — для отчёта и для соцсетей школы

### До конца дня
- [ ] **Внесли визит в воронку школ** ([68-forma-voronki-shkol.md](68-forma-voronki-shkol.md)): дата, класс, число детей/родителей, число анкет
- [ ] **Внесли все анкеты в CRM** с тегом «школа [id]»
- [ ] **Передали список** в отдел продаж для прозвона

---

## 📞 На следующий день

- [ ] **WhatsApp директору** с благодарностью + краткий отчёт (N анкет, готовы повторить)
- [ ] **Запросили обратную связь** у классрука по реакции детей
- [ ] **Проверили родчат** — пошли ли заявки через QR-код

## 📈 Через неделю

- [ ] **Сверка с РОП**: сколько анкет прозвонили, сколько записались на пробное, сколько дошли
- [ ] **Обновление воронки** школ — статус `в работе` → `закрыта-результат`
- [ ] **Договорённость с классруком** о повторном визите через 2–3 месяца

---

## ⛔ Что НЕ делаем — никогда

- ❌ Превышаем заявленное время (даже если идёт хорошо)
- ❌ Называем конкретную цену залу
- ❌ Критикуем других школ программирования / конкурентов
- ❌ Раздаём анкеты без галочки согласия ПДн
- ❌ Берём номер у ребёнка без родителя
- ❌ Обещаем гарантии («100% научим программировать», «гарантируем поступление»)
- ❌ Уходим без передачи благодарственного письма
- ❌ Задерживаемся в школе больше 5 минут после закрытия выступления

---

## 🎯 Метрики качества промо в ОУ

После каждого визита заполняем в воронке школ:

| Метрика | Норма МК | Норма родсобрания | Тревожный сигнал |
|---|---|---|---|
| Число анкет / число присутствующих | 30–50% (от класса + родчата) | 30–50% (от зала) | <20% |
| Валидность анкет (после прозвона) | 80%+ | 80%+ | <60% (промоутер «химичит» или давление было) |
| Доходимость анкеты → пробное | 60%+ | 60%+ | <40% |
| Доля «нет согласия ПДн» | 0 | 0 | Любое значение > 0 (нарушение!) |
