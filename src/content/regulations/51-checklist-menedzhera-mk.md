# 51. Чек-лист менеджера по скриптам 1–3

**Цель:** контроль качества звонков и доходимости на МК. Используется РОПом для прослушки.
**Связанные:** [47](47-skript-kasanie-1-zapis-na-mk.md) · [48](48-skript-kasanie-2-posle-mk.md) · [49](49-skript-kasanie-3-povtor.md) · [55 — Стоп-фразы](55-stop-frazy.md) · [56 — Стандарт общения](56-standart-obshheniya.md).

## Общие правила работы менеджера

- [ ] Использует формулировки скрипта **без изменений** сути и смысла.
- [ ] Фиксирует ответы родителя — слушает и слышит.
- [ ] Продаёт **результат для ребёнка** и **пользу для родителя**, а не формат.
- [ ] Говорит спокойно, уверенно, без оправданий.

## Скрипт №1 — Запись на МК

- [ ] Проговорено: «**не развлекуха на час**, а конкретный результат».
- [ ] Озвучена **формула МК** (игра + диагностика + персональный план).
- [ ] Проговорено **обязательное участие родителя** в МК.
- [ ] Проговорена польза **профтеста для родителя**.
- [ ] Проговорено, что результат ребёнка показывается **в конце урока**.
- [ ] Создан **дефицит** (осталось 1–2 места).
- [ ] Получено **обязательство прийти** («Да, хорошо»).
- [ ] Проговорена ответственность: тьютор и ассистент приглашаются **под ребёнка**.
- [ ] Проговорена фраза «не подводите нас».
- [ ] Проговорены **три касания перед МК**.

## Скрипт №2 — Первый звонок после МК

- [ ] Зафиксирован результат ребёнка (создал игру, прошёл диагностику).
- [ ] **Родитель сам проговорил ценность** (отвечая на «что больше всего понравилось»).
- [ ] Результат связан с запросом родителя из первого касания (логика / уверенность / усидчивость / интерес).
- [ ] Использована **рекомендательная формулировка** («предлагаю **развить** интерес и навыки»).
- [ ] Предложено начать обучение и подобрать расписание.

## Скрипт №3 — Второй звонок после МК

- [ ] Задан прямой вопрос на решение.
- [ ] Использована формулировка «**Бронируем за вами место?**».

При «ДА»:
- [ ] Проговорён договор и подписание в школе.
- [ ] Озвучена сумма оплаты за месяц.
- [ ] Предложены **способы оплаты** (включая маткапитал, если уместно).
- [ ] Озвучено сопровождение менеджером.

При «НЕТ»:
- [ ] Заданы вопросы **строго по скрипту**.
- [ ] Менеджер **не комментирует** ответ родителя.

---

## Запрещено

- ✗ Менять формулировки с изменением структуры и сути.
- ✗ Добавлять комментарии от себя.
- ✗ Смягчать дефицит и обязательство.
- ✗ Уговаривать при отказе.

---

## Формат прослушки РОПа

Для каждого звонка ставит галочки. Итог:
- ≥ 90% галочек — отличный звонок;
- 75–89% — обратная связь, повторное обучение;
- < 75% — переслушать со скриптом, сделать отработку под запись.
