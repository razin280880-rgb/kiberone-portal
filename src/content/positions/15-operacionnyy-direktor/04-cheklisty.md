# Чек-листы операционного директора

Все чек-листы — рабочий инструмент. Заполнение фиксируется (галочка с датой), хранится минимум 24 месяца (в роли ОД срок хранения дольше — для аналитики).

**Логика:** ОД координирует ~88 человек через ~10 ключевых руководителей. Без чек-листов либо тонет в задачах, либо теряет ключевые управленческие функции.

---

## ЧЕК-ЛИСТ 1. Ежедневный (заполняет в конце дня)

**Дата:** _______ **ОД:** _______

### Управление
- [ ] Утренний обзор сделан (дашборды + чаты + эскалации)
- [ ] Все плановые встречи на сегодня — состоялись или перенесены явно
- [ ] Если были эскалации — все обработаны или есть статус

### Стратегия
- [ ] Минимум 30 минут на стратегическую/проактивную работу
- [ ] Если в работе стратегические инициативы — есть прогресс

### Связь с собственником
- [ ] Все эскалации к собственнику отправлены
- [ ] Если есть вопросы для синка с собственником — записаны

### Тревожные сигналы (см. `06-ruka-na-pulse.md`)
- [ ] Проверены тревожные сигналы за вчера
- [ ] Эскалировано к собственнику, если нужно

### Документы
- [ ] Заметки за день внесены в дневник ОД
- [ ] План на завтра составлен

**Если хотя бы одна галочка не стоит** — день не закрыт.

---

## ЧЕК-ЛИСТ 2. Недельный (заполняет в пятницу)

**Неделя с _____ по _____** **ОД:** _______

### Управление
- [ ] Понедельничная групповая планёрка с РФ проведена (1.5 часа)
- [ ] Все плановые 1-1 за неделю состоялись (минимум 5)
- [ ] Командировка (если по графику) проведена
- [ ] Дайджест собственнику отправлен в пятницу

### Операционные KPI сети
- [ ] Дашборд сети проверен 5 дней из 5
- [ ] Все «красные» сигналы — есть план или статус
- [ ] Все 7 городов проанализированы (выручка, прибыль, операционные KPI)

### Стратегия
- [ ] Стратегическая работа за неделю — минимум 5 часов
- [ ] Если в работе инициативы — есть прогресс

### Команда
- [ ] Если есть открытые конфликты — есть статус разрешения
- [ ] Если есть открытые наймы — есть прогресс

### Документы
- [ ] Дашборд недели заполнен
- [ ] Отчёт собственнику (дайджест) отправлен
- [ ] Заметки за неделю систематизированы

### Следующая неделя
- [ ] План на следующую неделю составлен
- [ ] Все встречи в календаре

---

## ЧЕК-ЛИСТ 3. Ежемесячный (последняя пятница)

**Месяц _____** **ОД:** _______

### Финансы (главный блок)
- [ ] Сводный P&L всей сети закрыт к 7-му числу следующего месяца
- [ ] Выручка vs план: ___ %
- [ ] Прибыль vs план: ___ %
- [ ] ФОТ ≤ 50% от выручки: ___ %
- [ ] Если отклонения > 10% — есть план корректировки

### Операционные KPI сети
- [ ] База учеников: ___ (тренд — рост/падение)
- [ ] Месячный отток: ___ %
- [ ] Лидген (анкеты по сети): ___
- [ ] Конверсия пробное → продажа: ___ %
- [ ] Доля голосовых звонков фермеров: ___ %
- [ ] Качество (балл методиста): ___

### Управленческая команда
- [ ] Все 7 РФ получили 1-1 в этом месяце
- [ ] Все 5 центральных функций получили 1-1
- [ ] Если есть РФ / руководитель в зоне риска — есть план

### Стратегические инициативы
- [ ] Минимум 1 стратегическая инициатива — есть прогресс
- [ ] Презентация результатов собственнику подготовлена
- [ ] Стратегическая сессия с собственником прошла

### Самооценка
- [ ] Заполнила мини-опрос для себя: что получилось, что — нет

## ЧЕК-ЛИСТ 4. Понедельничная групповая планёрка с РФ (1.5 часа)

**Дата:** _____ **Участники:** _____

### Подготовка (за день до)
- [ ] Сводка KPI сети за прошлую неделю готова
- [ ] Повестка планёрки сформулирована (3-4 темы)
- [ ] Презентация (если нужна) — подготовлена

### Структура планёрки
- [ ] Начало вовремя
- [ ] Открытие, повестка (5 минут)
- [ ] Каждый РФ — итоги недели + приоритеты (по 4-5 минут × 7 = 30 минут)
- [ ] Обсуждение общих тем (30 минут)
- [ ] Закрытие, договорённости (15 минут)
- [ ] Завершение вовремя

### Что НЕ делать
- [ ] НЕ разбираю конкретные кейсы (это в 1-1)
- [ ] НЕ критикую РФ при остальных
- [ ] НЕ затягиваю > 1.5 часов

### После
- [ ] Протокол планёрки сохранён в общем чате
- [ ] Все договорённости — поставлены в трекер задач

## ЧЕК-ЛИСТ 5. 1-1 с РФ (раз в 2 недели)

**Дата:** _____ **РФ:** _____ **Город:** _____ **Длительность:** 45-60 минут

### Подготовка
- [ ] Прочитала дашборд филиала
- [ ] Знаю текущие KPI (выручка, прибыль, операционные)
- [ ] Готова к открытому разговору

### Структура встречи

**1. Дела филиала (15-20 минут)**
- [ ] Спросила о статусе главных приоритетов
- [ ] Что хорошо
- [ ] Что плохо
- [ ] Что нужно от меня (бюджет, решения, поддержка)

**2. Команда (10-15 минут)**
- [ ] Кто в зелёной/жёлтой/красной зоне
- [ ] Есть ли сложности с конкретными людьми

**3. Стратегия города (10 минут)**
- [ ] Прогресс по стратегическим инициативам в городе
- [ ] Гипотезы для тестирования

**4. Личное развитие РФ (10 минут)**
- [ ] Чему хочет научиться
- [ ] Какая поддержка нужна

**5. Договорённости (5 минут)**
- [ ] Конкретные действия с обеих сторон, с датами

### После встречи
- [ ] Записала в дневник ОД
- [ ] Если есть запросы — поставила задачу
- [ ] Через неделю — короткий чек-ин по результатам

## ЧЕК-ЛИСТ 6. 1-1 с центральной функцией (методист, РОП, РОС, маркетолог, HR)

**Дата:** _____ **Функция:** _____ **Длительность:** 45-60 минут

### Структура

**1. Главные показатели функции (10-15 минут)**
- [ ] Спросила о ключевых KPI
- [ ] Понимаю, что хорошо, что плохо

**2. Взаимодействие с филиалами (10-15 минут)**
- [ ] Какие конфликты или сложности с РФ
- [ ] Где нужна моя помощь как ОД

**3. Стратегические задачи функции (10 минут)**
- [ ] Прогресс по инициативам этой функции
- [ ] Запросы / предложения

**4. Личное развитие (10 минут)**
- [ ] Чему хочет научиться, куда расти

**5. Договорённости (5 минут)**

### После встречи
- [ ] Заметки в дневник ОД
- [ ] Договорённости — в трекер

## ЧЕК-ЛИСТ 7. Командировка в город (раз в квартал, 1-2 дня)

**Город:** _____ **Даты:** _____

### ДО командировки
- [ ] Согласована с РФ (минимум за 7 дней)
- [ ] Бронирование (билеты, гостиница) оформлено
- [ ] Сформулирован фокус командировки (3-5 пунктов)
- [ ] Запрошена информация: расписание уроков, состав команды, текущие проблемы

### День 1
- [ ] Встреча с РФ (1.5-2 часа): сводка ситуации
- [ ] Посещение 2-3 уроков
- [ ] 1-1 с тьюторами / администратором / менеджером развития (по 30 минут × 3-4 человека)

### День 2 (если есть)
- [ ] 1-1 с РФ (глубокое обсуждение стратегии города)
- [ ] Если есть внешние партнёры (директор школы и т.п.) — личная встреча
- [ ] Финальная встреча с РФ — договорённости с дедлайнами

### ПОСЛЕ
- [ ] Отчёт собственнику (1-2 страницы) в течение 3 дней
- [ ] Договорённости с РФ внесены в общий трекер
- [ ] Команда филиала получила письменное summary встречи

## ЧЕК-ЛИСТ 8. Месячный синк с собственником (1.5-2 часа)

**Дата:** _____ **Месяц:** _____

### Подготовка (за 24 часа)
- [ ] Презентация готова
- [ ] Все KPI сети зафиксированы
- [ ] Список «нужно решений от собственника» — составлен (3-5 пунктов)

### Структура встречи

**1. Финансы и KPI (20 минут)**
- [ ] P&L сети
- [ ] Главные операционные KPI
- [ ] Тренды

**2. По городам (20 минут)**
- [ ] Топ-3 города (что хорошо)
- [ ] Топ-3 проблемных города (что плохо, что планирую)

**3. Главное за месяц (15 минут)**
- [ ] Что хорошо
- [ ] Что плохо
- [ ] Что неожиданно

**4. Открытые вопросы (20 минут)**
- [ ] Эскалации к собственнику
- [ ] Предложения / гипотезы

**5. Стратегия (15 минут)**
- [ ] Прогресс по стратегическим инициативам
- [ ] Гипотезы / новые идеи

**6. План следующего месяца (10 минут)**

### После встречи
- [ ] Протокол отправлен собственнику в течение 24 часов
- [ ] Все договорённости — в работу

## ЧЕК-ЛИСТ 9. Стратегическая сессия с собственником (раз в квартал, полдня)

**Дата:** _____ **Квартал:** _____

### Подготовка (за 1-2 недели)
- [ ] Глубокий аналитический материал (10-20 страниц)
- [ ] Финансовые прогнозы на 6-12 месяцев
- [ ] Сценарии (3-5 главных стратегических развилок)
- [ ] Внешний контекст (рынок, конкуренты, тренды)

### Структура сессии (4 часа)

**1. Срез ситуации (45 минут)**
- [ ] Где сеть сейчас
- [ ] Куда движется (тренды)

**2. Финансовые прогнозы (45 минут)**
- [ ] Прогноз выручки и прибыли
- [ ] Сценарии (базовый, оптимистичный, пессимистичный)

**3. Стратегические развилки (1.5 часа)**
- [ ] 3-5 главных решений на следующий период
- [ ] Аргументы за / против каждого

**4. Решения и план (45 минут)**
- [ ] Согласование стратегических решений
- [ ] Декомпозиция на квартал

### После
- [ ] Протокол сессии — в письменном виде, согласован с собственником
- [ ] План на квартал утверждён
- [ ] Управленческая команда информирована (на следующей групповой планёрке)

## ЧЕК-ЛИСТ 10. Найм ключевой роли (РФ, центральная функция)

**Должность:** _____ **Кандидат:** _____

### Подготовка
- [ ] Согласована вакансия с собственником
- [ ] Описание соответствует [`../`](../) (соответствующая папка должности)
- [ ] HR подключён

### Финальное собеседование (с участием ОД)
- [ ] Прочитала резюме до встречи
- [ ] Подготовила 5-7 вопросов под роль
- [ ] Длительность не менее 60 минут (для РФ — 90 минут)

### Решение
- [ ] Согласовано с собственником
- [ ] Согласовано с центральной функцией (методист — для РФ)
- [ ] Оффер сделан в срок

### После найма
- [ ] Запущена программа онбординга (см. соответствующую папку)
- [ ] Назначен наставник
- [ ] Все доступы готовы

## ЧЕК-ЛИСТ 11. Аттестация управленческой команды (раз в полгода)

### Подготовка
- [ ] Каждый ключевой руководитель — KPI замерены за период
- [ ] Опрос команды для каждого руководителя (анонимно через HR)
- [ ] Самооценка каждого

### Сама аттестация
- [ ] 1-1 с каждым руководителем (1.5-2 часа)
- [ ] Обзор результатов
- [ ] Обсуждение зон роста
- [ ] Решения по премиям и зарплате

### После
- [ ] Отчёт собственнику по итогам аттестации
- [ ] Решения по зарплатам / премиям внедрены

## КАК ХРАНЯТСЯ ЧЕК-ЛИСТЫ

**Где:**
- Google Drive: `KIBERone Management / Чек-листы ОД / [год] / [месяц] /`
- Каждый чек-лист — отдельный файл с датой
- Раз в квартал — архивируется

**Доступ:**
- ОД — пишет
- Собственник — читает все
- HR — читает аттестации

**Срок хранения:**
- Ежедневные / недельные — 12 месяцев
- Месячные / квартальные — 24 месяца
- Найм / увольнение / аттестации — бессрочно
- 1-1 встречи — 12 месяцев

**Правило:** если чек-лист не заполнен или потерян, считается, что соответствующая работа НЕ была сделана.
