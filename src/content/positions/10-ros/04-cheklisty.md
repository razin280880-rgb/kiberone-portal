# Чек-листы РОС

Все чек-листы — рабочий инструмент. Заполнение фиксируется (галочка с датой), хранится минимум 12 месяцев.

**Логика:** РОС управляет 5 фермерами и работает со скриптами, которые меняются. Без чек-листов он либо тонет в оперативке, либо пропускает шлифовку скриптов / прослушивания.

---

## ЧЕК-ЛИСТ 1. Ежедневный (заполняет в конце дня)

**Дата:** _______ **РОС:** _______

### Управление командой
- [ ] Утренняя планёрка с фермерами проведена (15-20 минут)
- [ ] Чат с фермерами проверен 2-3 раза за день
- [ ] Эскалации от фермеров обработаны или в работе

### Прослушивание и обучение
- [ ] Прослушано минимум 8-12 звонков за день
- [ ] Каждому фермеру отправлен фидбэк по минимум 1-2 звонкам
- [ ] Если есть критическая ошибка — личный звонок фермеру

### Аналитика
- [ ] Дашборд удержания проверен утром (отписавшиеся за вчера, тренд)
- [ ] Если удержание упало — есть гипотеза почему

### Координация
- [ ] Чат с РОП / методистом / маркетологом / РФ проверен
- [ ] Если поступил вопрос — отвечен в течение 4 часов

### Документы
- [ ] Заметки за день внесены в дневник РОС
- [ ] План на завтра составлен

**Если хотя бы одна галочка не стоит** — день не закрыт.

## ЧЕК-ЛИСТ 2. Недельный (заполняет в пятницу)

**Неделя с _____ по _____** **РОС:** _______

### Прослушивание и фидбэк
- [ ] Каждый фермер прослушан минимум 8-10 раз за неделю
- [ ] Каждый получил письменный фидбэк
- [ ] 1-1 встречи: минимум с 2-3 фермерами за неделю

### Скрипты
- [ ] Если были обновления — обучены фермеры
- [ ] Если есть гипотеза по новой формулировке — протестирована в ролёвках

### Аналитика
- [ ] Удержание по каждому фермеру за неделю — посчитано
- [ ] Распределение фермеров по зонам (зелёная / жёлтая / красная) — обновлено
- [ ] Топ-3 типа ухода за неделю — выделены

### Координация
- [ ] Синк с РОП — был или запланирован
- [ ] Синк с методистом — если были учебные жалобы
- [ ] Если есть жалобы на качество новых клиентов от хантеров — переданы РОП

### Большая пятничная планёрка
- [ ] Подготовлена и проведена
- [ ] Топ-успехи и топ-провалы озвучены
- [ ] План на следующую неделю объявлен

### Документы
- [ ] Дашборд недели заполнен
- [ ] Отчёт за неделю отправлен собственнику / ОД

## ЧЕК-ЛИСТ 3. Ежемесячный (последняя пятница)

**Месяц _____** **РОС:** _______

### Метрики (см. `05-pokazateli-i-istochniki.md`)
- [ ] Месячный отток: ___% (норма ≤ 8%, целевое ≤ 5-6%)
- [ ] Доля голосовых звонков по отделу: ___% (норма ≥ 70%)
- [ ] Возврат после этапа 2 пропусков: ___% (норма ≥ 70%)
- [ ] Возврат после этапа 3: ___% (норма ≥ 40%)
- [ ] Доля «потерянных без причины»: ___% (норма ≤ 10%)
- [ ] Допродажи: Иннополис ___, лагерь ___
- [ ] Распределение фермеров: 🟢 ___, 🟡 ___, 🔴 ___

### Сезонные KPI
- **Если апрель-май:** конверсия продления: ___% (норма ≥ 75%)
- **Если июнь-август:** летний переход: ___% (норма ≥ 25%, целевое 50%)

### Аттестация фермеров
- [ ] Если в этом месяце была аттестация — проведена
- [ ] Решения по красной зоне приняты (план развития или замена)

### Скрипты
- [ ] Если были обновления — задокументированы
- [ ] Версия скриптов в общей папке актуализирована

### Координация
- [ ] Минимум 1 разбор «упавших в первый месяц» с РОП
- [ ] Минимум 1 разбор учебных жалоб с методистом

### Презентация собственнику / ОД
- [ ] Презентация месяца подготовлена
- [ ] Встреча проведена
- [ ] План на следующий месяц согласован

### Когортный анализ (раз в квартал)
- [ ] Если квартал заканчивается — проведён когортный анализ:
  - Что с купившими 3 месяца назад
  - Что с возвращёнными после красного флага

## ЧЕК-ЛИСТ 4. Качество звонка фермера (главный чек-лист прослушивания)

**Дата:** _____ **Фермер:** _____ **Тип звонка:** знакомство / пропуск этап 2 / пропуск этап 3 / продление / лагерь / exit-интервью **РОС:** _____

### Структура звонка (3 пункта)
1. [ ] Фермер представился по стандарту, имя ребёнка / родителя названо
2. [ ] Удержание внимания в первые 30 секунд (нет «холодного» открытия)
3. [ ] Логичная структура: вступление → выявление ситуации → решение → закрытие на действие

### Эмпатия и слушание (4 пункта)
4. [ ] Фермер слушал больше, чем говорил (норма: родитель говорит 60%+ времени)
5. [ ] Задавал открытые вопросы (не только «да/нет»)
6. [ ] Реагировал на эмоции родителя (не игнорировал)
7. [ ] Использовал имя ребёнка в значимых моментах

### Скрипт (3 пункта)
8. [ ] Соблюдал актуальный скрипт (не «отсебятина»)
9. [ ] Если возражение — отработал по скрипту (или достойно если выходил за скрипт)
10. [ ] Закрытие: конкретная договорённость с датой (не «подумайте»)

### Итого баллов: ___ / 10

**Шкала:**
- 9-10 — отличный звонок, эталон для команды
- 7-8 — нормальный, есть зоны роста
- 5-6 — есть проблемы, нужен план развития
- ≤ 4 — критический звонок, эскалация

**Заметки РОС:**
_______________________________________________

**Фидбэк фермеру отправлен:** [ ] Да, в _______ [ ] Нет

## ЧЕК-ЛИСТ 5. 1-1 с фермером (раз в 2 недели)

**Дата:** _____ **Фермер:** _____ **Длительность:** 30-45 минут

### Подготовка (10 минут)
- [ ] Прочитал статус фермера (KPI за 2 недели)
- [ ] Просмотрел оценки прослушанных звонков
- [ ] Подготовил 2-3 темы для обсуждения

### Структура встречи

**1. Как дела (5-10 минут)**
- [ ] Спросил, как чувствует себя в работе
- [ ] Слушал, не «руководил»

**2. Сложные кейсы (10-15 минут)**
- [ ] Какие клиенты «висят» (особенно в красной зоне)
- [ ] Что не получается
- [ ] Совместный разбор стратегии

**3. Обратная связь по звонкам (10-15 минут)**
- [ ] Конкретные наблюдения с прослушанных звонков
- [ ] Похвалил за конкретные удачные моменты
- [ ] Указал на 1-2 зоны роста (не больше)

**4. Развитие (5-10 минут)**
- [ ] Что хочет улучшить
- [ ] Что нужно от меня

**5. Договорённости (5 минут)**
- [ ] Конкретные действия с обеих сторон, с датами

### После встречи
- [ ] Записал в дневник
- [ ] Если есть запросы — поставил себе задачу

## ЧЕК-ЛИСТ 6. Аттестация фермера (раз в полугодие)

**Фермер:** _____ **Дата:** _____ **Стаж:** ___ месяцев

### Метрики за 6 месяцев
- [ ] Месячный отток: ___% (норма ≤ 8%)
- [ ] Доля голосовых звонков: ___% (норма ≥ 70%)
- [ ] Возврат после этапа 2: ___% (норма ≥ 70%)
- [ ] Конверсия продления (если апрель-май попадал): ___% (норма ≥ 75%)
- [ ] Допродажи (Иннополис + лагерь): ___ за 6 мес
- [ ] Средний балл звонков по чек-листу 4: ___ /10 (норма ≥ 7.5)
- [ ] Жалоб от родителей: ___ (норма ≤ 1 в квартал)

### Качественная оценка
- [ ] Соблюдает скрипты
- [ ] Растёт по фидбэкам (не повторяет одни и те же ошибки)
- [ ] Хороший командный игрок

### Самооценка фермера
- [ ] Фермер заполнил мини-опрос: что получается, что хочет улучшить

### Решение
- [ ] **Зелёная зона:** продолжаем, рассмотреть рост (старший фермер / повышение)
- [ ] **Жёлтая зона:** план развития на 3 месяца + контрольная аттестация
- [ ] **Красная зона:** план «исправить или замена» — 1 месяц

**Подписи:** РОС _____, собственник (для красной зоны) _____

## ЧЕК-ЛИСТ 7. Обновление скрипта

**Скрипт:** _____ **Дата:** _____ **Причина:** _____

### Подготовка
- [ ] Проблема ясно сформулирована (например: «60% клиентов отказываются на возражении 'дорого' при продлении»)
- [ ] Прослушано минимум 20 звонков с этой ситуацией
- [ ] Гипотеза новой формулировки готова

### Тестирование
- [ ] Ролёвки с командой (минимум 5)
- [ ] Корректировка по результатам
- [ ] Пилот: 2-3 фермера тестируют в реальных звонках 1 неделю
- [ ] Замер результата (конверсия с этой ситуацией)

### Внедрение
- [ ] Если пилот успешен — обучение всей команды
- [ ] Старая версия скрипта в архив, новая — в общую папку
- [ ] Обновлены все связанные материалы

### Контроль
- [ ] Через 2 недели — замер: помогает ли новый скрипт
- [ ] Если не помогает — откат или новая итерация

**Заполняется на КАЖДОГО уходящего клиента** — без исключений. Триггер: клиент заявил об уходе, перестал ходить ≥ 2 занятий подряд, попросил расторгнуть договор.

**Клиент:** _____ **Город / филиал:** _____ **Стаж клиента:** ___ мес **Фермер:** _____ **РОС:** _____ **Дата:** _____

### Шаг 1. Прослушивание последнего звонка фермера

- [ ] **Звонок был сделан?** [ ] Да, дата: _____ [ ] Нет (только переписка / нет контакта)

### Шаг 2. Анализ звонка (если был)

- [ ] Тип причины:
  - [ ] Реальная (переезд / финансы / ребёнок против / здоровье)

  - [ ] Сделанная хантером на входе («пуш-продажа») → эскалация РОП

### Шаг 3. Решение и действие

## КАК ХРАНЯТСЯ ЧЕК-ЛИСТЫ

**Где:**
- Google Drive: `KIBERone Management / Чек-листы РОС / [год] / [месяц] /`
- Каждый чек-лист — отдельный файл с датой
- Раз в квартал — архивируется

**Доступ:**
- РОС — пишет
- Собственник / ОД — читают
- HR — читает аттестации
- Фермеры — видят свои собственные оценки звонков и аттестации

**Срок хранения:**
- Ежедневные / недельные — 12 месяцев
- Месячные / когортный анализ — 24 месяца
- Аттестации — бессрочно
- Оценки звонков — 12 месяцев

**Правило:** если чек-лист не заполнен или потерян, считается, что соответствующая работа НЕ была сделана.
