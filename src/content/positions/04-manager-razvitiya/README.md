# Менеджер по развитию — рабочая папка должности

**Кто такой менеджер по развитию:** локальный сотрудник в одном из 7 филиалов (Челны / Нижнекамск / Казань / Елабуга / Краснодар / Сургут / Пермь). Главное звено лидогенерации в своём городе. Не продаёт сам — собирает анкеты потенциальных клиентов и передаёт в отдел продаж.

**Зона ответственности:** один город, его школы, его мероприятия, его промоутеры, его поток анкет.

**Главный продукт менеджера по развитию:** **стабильный поток квалифицированных анкет** в город — измеряется в количестве анкет за месяц и доходимости «анкета → пробное занятие».

**Норма потока анкет (зависит от города):**
- Челны / Нижнекамск / Казань: 200-300 анкет/мес
- Елабуга: 100-150 анкет/мес
- Краснодар / Сургут / Пермь: 150-250 анкет/мес (при росте — больше)

## Кому подчиняется

| Город | Подчинение |
|---|---|
| Где есть руководитель филиала | Руководителю филиала (методически), маркетологу (центральному) — по плану |
| Где руководителя филиала нет | Напрямую собственнику + центральному маркетологу |

**Параллельно работает с:**
- Маркетологом (центральным) — материалы, лендинги, реклама
- РОПом и хантерами (центральными) — куда передаются анкеты
- Промоутерами своего города (2-5 человек, нанимаются и контролируются ею)
- Тьюторами своего города — для проведения мастер-классов

## Как пользоваться этой папкой

**Если нанимаете нового менеджера в новый город:**
1. Дать прочитать `01-rol-i-funkcii.md` ДО собеседования
2. После найма — запустить программу `02-sistema-adaptacii.md`
3. Через месяц — оценить по `07-kpi-i-mesyachnye-metriki.md`

**Если менеджер уже работает (старый сотрудник):**
1. Сверить факт с `01-rol-i-funkcii.md` — что делает по факту, что должен делать
2. Проверить чек-листы — выполняет ли регулярные процедуры
3. Запустить замер по `05-pokazateli-i-istochniki.md` за последние 1-3 месяца
4. По результатам — корректировка зоны ответственности или ввод KPI с премией

**Если в городе нет менеджера по развитию (функцию закрывает руководитель филиала или никто):**
- Срочность: высокая. Без выделенного менеджера лидген хаотичен, сезонность бьёт сильнее, рост невозможен
- Особенно критично для Краснодара / Сургута / Перми (все три проблемных филиала — там нет полноценной лидген-функции)
- Документы 01 и 02 — основа для найма

## Особенность роли в KIBERone-сети

В отличие от методиста (одного на всю сеть), менеджеров по развитию **должно быть 7 — по одному в каждом городе**. Это значит:

- Стандарт работы — **общий** (эта папка)
- Тактика и контакты — **локальные** (свой паспорт города ведёт каждый)
- Эталон — Челны (там старая база партнёрств), копируем туда, где работает похожий рынок

**Где особенно сложно:**
- **Краснодар** — большой город, конкуренция, школы менее доступны → нужен опытный менеджер с опытом B2B-продаж в образовании
- **Сургут** — маленький город, узкое окно школ, сезонность ярче → менеджер должен быть «на земле», знать всех директоров лично
- **Пермь** — средний город, конкуренция за внимание родителей → акцент на мероприятия и контент

**Где легче:**
- **Челны** — домашняя база, узнаваемость → менеджер скорее «администратор» партнёрств, чем «охотник»
- **Нижнекамск / Елабуга** — маленькие города, достаточно 3-5 сильных школ
- **Казань** — большой город, но узнаваемость KIBERone в Татарстане выше → меньше препятствий
