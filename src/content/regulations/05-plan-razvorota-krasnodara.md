# План разворота филиала Краснодар

**Ситуация на 2026-04-29:**
- Филиал работает 4-5 лет
- Куплен у предыдущего франчайзи в сентябре 2025
- Динамика учеников: 250 (пик) → 180 (на момент покупки) → 120 (сейчас)
- **-33% за 6 месяцев** после покупки
- 3 локации, 40 учеников/локация
- Точка безубыточности: 200 учеников
- Текущий месячный убыток: ~460 тыс ₽
- Менеджера по развитию НЕТ
- Текучка тьюторов из-за низкой нагрузки

**Прогноз без вмешательства:**
- Через 6 месяцев: ~80 учеников, убыток ~700 тыс ₽/мес
- Через 12 месяцев: ~50 учеников, точка невозврата
- Накопленный убыток за год: **~7-8 млн ₽**

**Время для решения: 60-90 дней.** После этого спасение будет невозможно.

## 3 СЦЕНАРИЯ — выбор

### Сценарий А: «Антикризис-разворот»
**Цель:** удержать филиал, выйти на BEP за 12-18 месяцев, через 24 месяца — в плюс.

**Структура действий:**
1. Закрыть 2 слабые локации, оставить 1 опорную
2. Срочно нанять менеджера по развитию (МР)
3. Внедрить процессы удержания (отчёт прогресса, регламент пропусков — см. отдельные документы)
4. Лично возглавить разворот: ездить в Краснодар 1-2 раза в месяц первые 6 месяцев
5. Снизить ФОТ за счёт сокращения штата с локаций

**Инвестиции:** ~2-3 млн ₽ (МР зарплата 6 мес + маркетинг + ваше время)
**Риск:** средний. Сценарий рабочий при условии вашего личного фокуса.

### Сценарий Б: «Контролируемое сжатие»
**Цель:** сжать филиал до прибыльной 1 локации, оставить как «лабораторию» Краснодарского рынка для будущего масштабирования.

**Структура действий:**
1. Закрыть 2 локации
2. Не нанимать МР
3. Сжаться до 60-80 учеников на 1 локации
4. Работать на самоокупаемость
5. Через 6 месяцев — пересмотр: либо перезапуск с инвестициями, либо выход

**Инвестиции:** минимальные (расходы на закрытие)
**Риск:** низкий. Получите ясность через 6 месяцев.

### Сценарий В: «Выход»
**Цель:** продать или закрыть филиал, освободить ресурс для других городов.

**Структура действий:**
1. Найти покупателя (другого франчайзи KIBERone, готового взять «проблемный»)
2. Если не найдётся за 30 дней — закрыть с возвратом денег действующим клиентам
3. Освободить ваше время и кэш для развития 4 сильных + Сургута/Перми

**Инвестиции:** возврат клиентам + штрафы по аренде ~1-2 млн ₽
**Риск:** низкий, но необратимо.

## 🎯 РЕКОМЕНДАЦИЯ: СЦЕНАРИЙ А (Антикризис-разворот)

**Почему не Б:**
- Сжатие без МР = продолжение падения, просто медленнее
- «Лаборатория» — это для зрелого бизнеса, у вас сейчас нет ресурса экспериментировать

**Почему не В:**
- Краснодар — миллионник, рынок есть. Закрытие = признание поражения
- Для бренда KIBERone (вы франчайзи!) закрытие города — репутационный удар
- В случае разворота через 24 месяца филиал может стать прибыльным активом 1-2 млн ₽/мес

**Почему А несмотря на риск:**
- Один найм (МР) + закрытие 2 локаций = относительно низкая инвестиция
- У вас сейчас 250 → 120: значит **в этом городе работали системы, которые вытаскивали филиал на 250**. Нужно их восстановить, а не строить с нуля
- Анти-кризис для вас лично также образовательный — отрабатываете схему, которую сможете применить в других проблемных городах

## ДОРОЖНАЯ КАРТА — 90 дней

### 🔥 Дни 1-7 (срочно, эта неделя)

- [ ] **Аудит на месте.** Сами или помощник-директор едет в Краснодар, проводит:
  - Встречу с тьюторами (что они видят, что говорят про отток)
  - Встречу с действующими родителями, которые остались (4-5 семей) — почему остались
  - Встречу с родителями, которые ушли (5-7 семей по последним 3 месяцам) — реальная причина
  - Аудит 3 локаций: трафик, состояние, аренда, расположение
- [ ] **Решение по локациям.** На основе аудита: какие 2 закрываем, какая 1 — опорная
- [ ] **Запуск вакансии менеджера по развитию** (см. отдельный документ или создайте по аналогии с РОС)

### 🟠 Дни 8-30

- [ ] **Закрытие 2 локаций.** Уведомление родителям, перенос в опорную локацию, переговоры с арендодателями
- [ ] **Перевод тьюторов.** Из 6 тьюторов на 3 локациях → 3-4 тьютора на 1 локации с большей нагрузкой → большая зарплата → стабильность
- [ ] **Найм МР.** Финал собеседования вы проводите лично
- [ ] **Внедрение Отчёта прогресса** (см. отдельный документ) — пилот именно с Краснодара
- [ ] **Личный визит:** вы или помощник-директор едет на 2-3 дня в Краснодар, проводите встречу с командой

### 🟡 Дни 31-60

- [ ] **МР начинает работать:** идёт по школам, организует мастер-классы, собирает анкеты
- [ ] **Внедрение регламента пропусков** (см. отдельный документ) — отдельный фокус на Краснодарском фермере
- [ ] **Запуск летней программы для текущих** — Краснодар получает приоритетную поддержку (см. бизнес-кейс лета)
- [ ] **Метрика:** база перестала падать (ноль чистого оттока в месяц как минимум)

### 🟢 Дни 61-90

- [ ] **Видим первые анкеты от МР** — конверсия в пробный, в платного
- [ ] **Метрика:** база начала расти (+5-10 учеников)
- [ ] **Решение о следующем этапе:**
  - Если динамика положительная → продолжаем антикризис, ждём BEP к месяцу 12-15
  - Если динамика прежняя (продолжает падать) → переход на сценарий Б (сжатие до 1 локации, без МР)
  - Если резко вверх → начинаем планировать вторую локацию через 6-9 месяцев

## КОММУНИКАЦИЯ С КЛИЕНТАМИ ПРИ ЗАКРЫТИИ ЛОКАЦИЙ

### Что говорим (родителям закрывающихся локаций)

```
Уважаемые родители!

С [дата] мы оптимизируем работу филиала Краснодар — переводим обучение в нашу 
центральную локацию по адресу [адрес опорной локации]. Это позволит:

✓ Сохранить ваших тьюторов в одном месте
✓ Расширить расписание (больше слотов на выбор)
✓ Улучшить инфраструктуру обучения

Расписание занятий мы сохраняем — переход полностью бесшовный. 
Если новая локация далеко — обсудим индивидуально, найдём решение 
(скидка / онлайн-формат / возврат за оставшийся период).

С вопросами — звоните [фермер] +7 XXX XXX XX XX.
```

### Что НЕ говорим
- Не упоминаем «закрытие», «убыточность», «оптимизация затрат»
- Не извиняемся как за провал — это рутинное бизнес-решение
- Не предлагаем массовый возврат — это спровоцирует уход

### Ожидаемая потеря на закрытии
- Из 80 учеников на 2 закрываемых локациях: 60-70% перейдут в опорную
- Потеря: ~25-30 учеников = ~150-175 тыс ₽/мес
- Но: -2 аренды + -2 коммуналки + сокращение тьюторов + сокращение администрирования = экономия ~400-500 тыс ₽/мес
- **Чистый эффект закрытия: +200-300 тыс ₽/мес** уже с месяца 2

## ВАШИ ЛИЧНЫЕ ОБЯЗАТЕЛЬСТВА

**Это самый важный пункт всего плана.**

Ни один разворот удалённого филиала **не работает**, если собственник не выкладывается лично. Чтобы план сработал, вам нужно:

- [ ] **Минимум 1 поездка в Краснодар в первый месяц** (3-4 дня)
- [ ] **Минимум 1 поездка в каждый из месяцев 2-6** (2-3 дня)
- [ ] **Еженедельная личная Zoom-встреча с МР** (45 мин)
- [ ] **Ежедневный взгляд на ключевые метрики Краснодара** (5 мин в день)

Если вы не готовы к такому фокусу — переходим к сценарию Б или В **сейчас**, не теряя кэш на полумерах.

Краснодар требует не «процессов и регламентов», а **личного присутствия собственника** в течение 6-12 месяцев. Это инвестиция вашего внимания.
