# РОР: KPI и ежемесячные метрики

KPI РОР, формула премии, ежемесячный дашборд для собственника.

---

## Базовый принцип премирования

РОР получает **оклад + переменную часть**. Переменная — главный мотиватор и должна быть существенной (≥ 50% от оклада при выполнении KPI).

### Структура зарплаты

| Часть | Размер | Триггер выплаты |
|---|---|---|
| **Оклад** | 100 000 – 150 000 ₽ | Стандартно, два раза в месяц |
| **% от плана по входным анкетам** | 0.5–2% от выручки нового бизнеса | Ежемесячно (премия за месяц) |
| **Квартальная премия** | 0.3–0.5 оклада | Раз в квартал при выполнении ≥ 90% KPI |
| **Годовая премия** | 1–2 оклада | Раз в год при выполнении ≥ 90% KPI |

**Целевая зарплата при норме:** 180–300 тыс ₽ в месяц.

---

## KPI РОР — 5 ключевых метрик

### KPI 1. Поток квалифицированных анкет (главный KPI)

**Метрика:** количество анкет в статусе «записан на пробное» за месяц.

**Вес в премии:** 40%.

**Целевое значение:** план на месяц (задаётся квартально, исходя из плана по выручке нового бизнеса).

**Шкала:**
| Факт vs план | Доля премии по KPI 1 |
|---|---|
| ≥ 110% | 120% |
| 100–110% | 100% |
| 90–99% | 75% |
| 80–89% | 50% |
| 70–79% | 25% |
| < 70% | 0% |

### KPI 2. Качество анкет

**Метрика:** доля «грязных» анкет, переданных в РОП за месяц.

**Вес в премии:** 20%.

**Целевое значение:** ≤ 10%.

**Шкала:**
| Доля грязных | Доля премии по KPI 2 |
|---|---|
| ≤ 5% | 120% |
| 5–10% | 100% |
| 10–15% | 75% |
| 15–20% | 50% |
| 20–25% | 25% |
| > 25% | 0% |

### KPI 3. Доходимость до пробного

**Метрика:** доля записавшихся, которые реально пришли на пробное за месяц.

**Вес в премии:** 15%.

**Целевое значение:** ≥ 70%.

**Шкала:**
| Доходимость | Доля премии по KPI 3 |
|---|---|
| ≥ 80% | 120% |
| 70–80% | 100% |
| 60–70% | 75% |
| 50–60% | 50% |
| < 50% | 0% |

### KPI 4. Диверсификация каналов

**Метрика:** **ни один канал не превышает 50% от всех анкет месяца** + минимум 4 активных канала на каждый город.

**Вес в премии:** 10%.

**Шкала (двоичная):**
- Выполнено по обоим условиям — 100%
- Нарушено одно условие — 50%
- Нарушены оба — 0%

### KPI 5. Управленческая ритмика

**Метрика:** комплексная — проведение планёрок, 1-1, обучений по нормам.

**Вес в премии:** 15%.

**Подкомпоненты:**
- Ежедневная планёрка: ≥ 95% рабочих дней
- Стратегическая планёрка: 100% посещение
- 1-1 с каждым менеджером развития: раз в 2 недели
- Еженедельная обучающая сессия: ≥ 90% запланированных в квартале
- Прослушивание звонков: ≥ 25 в неделю
- Аттестация: 100% сотрудников отдела имеют аттестацию ≤ 6 месяцев

**Шкала:**
| Сколько подкомпонентов выполнено | Доля премии по KPI 5 |
|---|---|
| 6/6 | 120% |
| 5/6 | 100% |
| 4/6 | 75% |
| 3/6 | 50% |
| < 3/6 | 0% |

## Расчёт ежемесячной премии — формула

```
Премия за месяц = (% от выручки нового бизнеса × выручка)
                  × ВЗВЕШЕННАЯ_ДОЛЯ_KPI
```

где **взвешенная доля KPI** = сумма всех KPI с их весами:

```
ВЗВЕШЕННАЯ_ДОЛЯ_KPI = 0.40 × KPI_1
                    + 0.20 × KPI_2
                    + 0.15 × KPI_3
                    + 0.10 × KPI_4
                    + 0.15 × KPI_5
```

### Пример расчёта

Допустим, в месяце:
- Выручка нового бизнеса = 5 000 000 ₽
- Базовый процент РОР = 1.5%
- KPI_1 (план анкет) = 100% выполнено → 100%
- KPI_2 (грязные ≤ 10%) = 7% → 100%
- KPI_3 (доходимость) = 65% → 75%
- KPI_4 (диверсификация) — выполнено → 100%
- KPI_5 (ритмика) — 5 из 6 → 100%

```
ВЗВЕШЕННАЯ_ДОЛЯ = 0.40×1.00 + 0.20×1.00 + 0.15×0.75 + 0.10×1.00 + 0.15×1.00
                = 0.40 + 0.20 + 0.1125 + 0.10 + 0.15
                = 0.9625 (96.25%)

Премия = 5 000 000 × 1.5% × 96.25%
       = 75 000 × 0.9625
       = 72 188 ₽
```

При окладе 130 000 ₽ → итог: **130 000 + 72 188 = 202 188 ₽ за месяц**.

## Квартальная премия

Дополнительная разовая выплата раз в квартал при условии:

- Все 3 месяца квартала — выручка нового бизнеса ≥ 95% плана
- Все 3 месяца — взвешенная доля KPI ≥ 85%
- Нет действующих письменных предупреждений по дисциплине

**Размер:** 0.3–0.5 оклада (по решению собственника).

---

## Годовая премия

Дополнительная разовая выплата раз в год при условии:

- Все 4 квартала — KPI выполнен ≥ 90%
- Выполнен годовой план по выручке нового бизнеса ≥ 95%
- Минимум 4 запущенных и масштабированных новых канала за год
- Доля «зелёных» менеджеров развития к концу года ≥ 70%
- Минимум 8 новых школьных партнёрств в сети за год

**Размер:** 1–2 оклада (по решению собственника).

## Депремирование (штрафы)

Сумма депремирования вычитается из переменной части месяца:

| Нарушение | Размер штрафа |
|---|---|
| Пропуск ежедневной планёрки без уведомления | 5% от премии месяца за каждый случай |
| Пропуск стратегической планёрки | 10% от премии месяца |
| Не проведена обучающая сессия в неделю без переноса | 5% от премии месяца |
| Доля «грязных» анкет > 25% в любую неделю | 15% от премии месяца |
| Жалоба от партнёра / школы, не отработанная в течение 48 часов | 10% от премии месяца |
| Менеджер развития не имеет 1-1 > 3 недель | 10% от премии месяца |
| Не сдан ежемесячный отчёт собственнику | 20% от премии месяца |

**Кумулятивно депремирование не может превышать 50% премии месяца.** При систематических нарушениях — внеочередная аттестация и решение о судьбе РОР.

## Ежемесячный отчёт для собственника / ОД

РОР сдаёт **до 3-го рабочего дня нового месяца** отчёт по шаблону:

### Раздел 1. Цифры (объёмные)

- Анкеты получено: всего / по городам / по каналам
- Записи на пробное: всего / план / факт
- Доходимость до пробного
- Конверсия квалификации
- Активные промо-точки и партнёрства

### Раздел 2. Качество

- Доля грязных анкет
- Заполняемость обязательных полей
- Средний балл звонков менеджеров

### Раздел 3. Финансы

- CAC по каналам
- Распределение бюджета по каналам
- Изменения vs прошлого месяца

### Раздел 4. Команда

- Состояние менеджеров развития (зоны)
- Состояние промоутеров
- Найм/увольнения

### Раздел 5. Выводы и решения

- 3 главные удачи месяца
- 3 главные провала месяца
- 3 решения на следующий месяц

### Раздел 6. Запросы к собственнику / ОД

- Что нужно от собственника
- Какие решения требуют согласования
- Какие ресурсы нужны (бюджет, найм, ИТ-инструменты)

## Связь KPI РОР с KPI других руководителей

| KPI | Связь с другими |
|---|---|
| Поток анкет | Прямо влияет на план РОП (конверсия → продажа) |
| Качество анкет | РОП теряет время на «грязных» — снижается его конверсия |
| Доходимость | Совместная зона с РОП (звонок-напоминание хантера) |
| Диверсификация каналов | Снижает риск для всей сети |
| Ритмика | Влияет на качество команды → косвенно на все остальные KPI |

**Кросс-KPI с РОП:** общая «конверсия анкета → договор». РОР отвечает за качество анкеты, РОП — за конверсию. Если общая конверсия падает — совместный разбор и совместная корректировка.
