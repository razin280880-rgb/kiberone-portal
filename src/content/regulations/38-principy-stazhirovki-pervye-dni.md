# Принципы стажировки: с первого дня — реальные задачи

## Главный принцип

**В KIBERone новый сотрудник с первого дня выполняет реальные задачи своей должности — звонит, договаривается, продаёт, преподаёт, ведёт переговоры. Только после того, как он показал свою силу в основной работе, мы переходим к изучению сервиса, CRM, проектов и других тонкостей.**

## Почему так

Стандартный подход на рынке: сначала неделя-две обучения → потом сотрудник садится за работу → выясняется, что не подходит. Мы делаем наоборот: сначала смотрим силу в сути работы, потом учим тонкостям.

**Зачем:**
- Экономим время — слабый отсеивается за 2–3 дня.
- Сотрудник сразу понимает, та ли это работа.
- Сильные быстрее выходят на результат.
- Не воспитываем «теоретиков» — людей, которые знают регламент, но не делают цифру.

## Что это НЕ значит

❌ Бросить в воду без подготовки → ✅ Дать реальную задачу + наставника + быструю обратную связь.
❌ Не объяснять ничего → ✅ Объяснить минимум для первого действия — остальное по ходу.
❌ Не давать права на ошибку → ✅ Право на ошибку есть. Нет права на повторение одной ошибки после обратной связи.

## По ролям в первые 3–5 дней

| Должность | Что делает с 1 дня |
|---|---|
| Хантер | Звонит по холодной базе, записывает на пробное |
| Фермер | Звонит сопровождаемой базе по конкретным задачам |
| Менеджер развития | Идёт в школу, договаривается с директором |
| Тьютор | Ассистирует на уроке + к 3–5 дню проводит часть урока сам |
| Ассистент | Помогает на уроке с первого занятия |
| Методист | Проверяет одного-двух тьюторов на открытых уроках с первой недели |
| Маркетолог | Делает один конкретный креатив / пост / запуск с первого дня |
| HR | Обзванивает кандидатов на открытые вакансии с первого дня |
| Руководитель филиала | Планёрка с командой + один визит в школу с первого дня |
| РОР / РОП / РОС | Личное участие в 5–10 разговорах подчинённых на первой неделе |

## Когда переходим к обучению сервису

После того как сотрудник показал, что справляется с базовой работой (обычно 1–2 недели). Признаки:
- Хантер записывает на пробное хотя бы 1/день
- Фермер отвечает на типовые запросы без подсказок
- Тьютор провёл 2+ урока самостоятельно с приемлемым отчётом
- Менеджер развития сам провёл результативный визит в школу
- HR провёл 5+ первичных интервью

После этого: глубокая CRM, программа KIBERone в деталях, внутренние сервисы, связки с другими отделами, метрики.

## Роль руководителя в первые дни

1. Одна конкретная задача в день — не «изучи всё».
2. Быть рядом — разбор после каждого действия.
3. Поправлять быстро, а не копить на еженедельный разбор.
4. Решение «оставляем / расстаёмся» — по итогам 3–5 дней реальной работы.

---

**Для HR:** обещай кандидату «с первого дня реальная работа + наставник», а не «две недели мягкой адаптации».
**Для руководителей:** твоя задача — давать задачу и быстро разбирать результат, а не «обучать».
**Для нового сотрудника:** твой первый день — это про реальное действие. Это нормально.
