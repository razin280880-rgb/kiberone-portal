# Фермер (менеджер по сопровождению) — рабочая папка должности

**Кто такой фермер:** менеджер отдела сопровождения, работает с **уже платящими клиентами** (родителями детей, которые ходят в KIBERone). Главная функция — удержание, отработка пропусков, продление, допродажи (Иннополис, летний лагерь, дополнительные курсы).

**Зона ответственности:**
- На сети сейчас 5 фермеров
- 2 фермера — каждый ведёт по одному городу
- 3 фермера — каждый ведёт по 2-3 города
- Целевая нагрузка: ~320-350 учеников на фермера

**Главный продукт фермера:** **сохранённый и продлённый клиент** — измеряется через долю удержания за месяц, долю продления на новый учебный год, долю допродаж.

---

## Кому подчиняется

**Сейчас (на 2026-05-01):**
- Руководителя отдела сопровождения **НЕТ**
- Утренние планёрки с фермерами проводит **сам собственник**
- Это создаёт перегрузку собственника и отсутствие профессионального наставничества

**Целевая структура:**
- Фермеры подчиняются РОС (руководителю отдела сопровождения) — см. [`../../04-tz-rukovoditel-otdela-soprovozhdeniya.md`](../../04-tz-rukovoditel-otdela-soprovozhdeniya.md)
- Контроль качества звонков делает отдельный сотрудник (уже есть)

**Параллельно работает с:**
- Тьюторами в своём городе — обратная связь от родителей, координация по пропускам/возвратам
- Методистом (центральным) — эскалация жалоб на учебный процесс
- Хантерами — приём клиентов после первой оплаты
- Контролем качества звонков — получение фидбэка по ошибкам в звонках

## Главная боль текущей оргструктуры (что эта папка пытается чинить)

🚩 **Фермеры по факту делают переписку вместо звонков.** Это снижает удержание на 30-40%, потому что:
- Родитель в переписке легче «отписывается» («извините, не получится продолжать»)
- В голосе слышно сомнения родителя — в переписке они спрятаны
- Эмоциональная связь не строится через текст
- Сложные возражения (продление, отписка) переписка не разруливает

🚩 **Скрипты «может быть не такие хорошие»** (со слов собственника). Пользователь делает их сам через AI, нет специалиста, который шлифует их по результатам реальных диалогов.

🚩 **Нет РОС.** Фермеры подчиняются собственнику напрямую → нет профессионального наставничества, нет регулярных разборов кейсов, нет роста.

🚩 **Контроль качества — реактивный, не обучающий.** Сотрудник КК слушает звонки, корректирует ошибки → но не обучает заранее.

**Что должна сделать эта папка:**
1. Закрепить голос как главный канал (через KPI, чек-листы, депремирование)
2. Дать фермеру **готовые скрипты** на каждый ключевой диалог + шаблоны для нестандартных
3. Создать **систему «рука на пульсе»** — что считается тревожным сигналом, как реагировать
4. Формализовать KPI с фокусом на удержание (а не только на количество звонков)

## Особенность роли в KIBERone-сети

В отличие от менеджера по развитию (1 на город) — фермеры **распределяются по нагрузке, а не по городам**. Один фермер может вести Сургут целиком (один город) или Пермь + Краснодар + Елабуга (три города).

**Это создаёт сложность:**
- Фермер «не на земле» — не знает тьюторов лично, не видит уроков
- Может «потерять» клиента, потому что не успевает физически пойти на встречу
- Зависит от качества данных в CRM

**Это решается так:**
- Минимум раз в квартал — командировка / выезд в каждый из закреплённых городов (на 1-2 дня)
- Видеосозвоны с тьюторами (для контекста)
- Чёткое распределение нагрузки: один фермер — не более 350 учеников, иначе теряется качество

## Норма нагрузки

| Параметр | Норма | Текущая ситуация (на 2026-05) |
|---|---|---|
| Количество учеников на фермера | ~320-350 | Соответствует |
| Количество звонков в день | ≥ 25 | Не замеряется системно |
| Доля голосовых vs переписки | ≥ 70% голос | 🚩 Сейчас перевёрнуто |
| Месячное удержание | ≥ 92% (отток ≤ 8%) | 🚩 Сейчас 7-8% — на нижней грани нормы |
| Продление на новый сезон | ≥ 75% | 🚩 Сейчас ~60% (норма не достигается) |

**Где основные проблемы:**
- Лето: 85-90% уходят (норма для рынка ~30-40%) — критичный провал
- Продление: 60% (норма 75%) — теряем 15% базы каждый сезон
- Месячный отток 7-8% — на грани нормы, в случае ухудшения = катастрофа
