# KIBERone — Портал команды

Внутренняя база знаний сети KIBERone: должности, регламенты, скрипты, онбординг.

## Что внутри

- **13 должностей** с описанием роли, KPI, чек-листов и плана адаптации
- **15 регламентов** процессов сети (Кибер-ярмарка, пропуски, аттестации, реанимация и т.д.)
- **20 скриптов** звонков и WhatsApp-шаблонов
- **Онбординг-маршрут** для нового сотрудника
- **Поиск** по всему порталу (Pagefind, работает на русском с морфологией)

## Источник контента

Все материалы — markdown-файлы из `C:\Users\Honor\kiberone-management`. Перед сборкой запускается `sync-content.mjs`, который копирует .md-файлы в `src/content/`. **Менять контент нужно в `kiberone-management`**, не в `src/content/`.

## Локальная разработка

```bash
npm install
npm run dev          # http://localhost:3005
```

## Сборка и проверка

```bash
npm run build        # собирает dist/ + индекс Pagefind
npm run preview      # локально проверяет production-сборку
```

## Деплой

Через Vercel: при `git push` в `main` Vercel автоматически собирает портал и публикует.

## Стек

- Astro 5 + Tailwind 3 + Typography
- Pagefind (статический поиск по контенту)
- Кастомный remark-плагин для переписывания внутренних .md-ссылок
