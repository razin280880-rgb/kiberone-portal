# Чек-листы тьютора

Все чек-листы — рабочий инструмент. Заполнение фиксируется (галочка с датой), хранится минимум 6 месяцев.

**Логика:** тьютор работает с 30-60 живыми детьми и проводит ~10 уроков в неделю. Чек-листы — страховка от того, что что-то «забыто» (подготовка, заполнение дневника, отчёт прогресса).

---

## ЧЕК-ЛИСТ 1. Подготовка к уроку (за 30-45 минут до)

**Дата:** _____ **Урок:** _____ **Группа:** _____ **Тема:** _____

### Содержание
- [ ] Изучен методический материал по теме
- [ ] Подготовлены примеры (минимум 2-3)
- [ ] Подготовлен главный проект урока
- [ ] Подготовлены 1-2 дополнительные задачи (для тех, кто быстро справится)
- [ ] Если новая тема — пробный прогон в голове

### Помещение и оборудование
- [ ] Помещение готово (чисто, проветрено)
- [ ] Компьютеры включены и проверены
- [ ] Проектор / экран работает
- [ ] Все материалы для детей подготовлены (если нужны)

### Команда
- [ ] Координация с ассистентом за 5-10 минут до урока
  - План урока озвучен
  - Особые дети упомянуты («Петя сегодня тяжелее обычного»)
  - Распределение зон работы
- [ ] Если есть стажёр или новичок-наставленный — он в курсе плана

### Себе
- [ ] Готов к уроку (не «утомлён» прошлым уроком)
- [ ] Если плохое настроение — собрался ради детей

## ЧЕК-ЛИСТ 2. Проведение урока (10 пунктов — главный, по которому оценивает методист)

**Дата:** _____ **Группа:** _____ **Тема:** _____

### Структура урока (3 пункта)
1. [ ] Начал с приветствия и постановки цели урока
2. [ ] Все дети были вовлечены в первые 5 минут
3. [ ] Объяснение темы простым языком, с примерами

### Работа с детьми (4 пункта)
4. [ ] Дал детям достаточно времени на самостоятельную работу
5. [ ] Помогал индивидуально, не «вёл за руку» всю группу
6. [ ] Реагировал на ошибки конструктивно (не «неправильно», а «попробуй ещё»)
7. [ ] Корректно работал с трудным ребёнком (если был такой эпизод)

### Управление уроком (1 пункт)
8. [ ] Управлял временем урока (не выпал из тайминга)

### Завершение (2 пункта)
9. [ ] Завершил урок рефлексией или показом результатов
10. [ ] Поддержал мотивацию (похвала, конкретная)

### Итого: ___ / 10

**Шкала (та же, что у методиста):**
- 9-10 — отличный урок
- 7-8 — нормальный, есть зоны роста
- 5-6 — есть проблемы, нужен план развития
- ≤ 4 — критический урок

**Самооценка тьютора:** ___ / 10

## ЧЕК-ЛИСТ 3. Послеурочная обратная связь (10 минут после урока)

### Дневник тьютора (Google Form)
- [ ] Внесена посещаемость (кто пришёл, кто пропустил)
- [ ] Внесено содержание урока (что прошли, что усвоилось плохо)
- [ ] Особые случаи зафиксированы (если были — ребёнок плакал / показал прогресс / поссорился)

### Координация с ассистентом
- [ ] Короткий разбор урока (3-5 минут)
- [ ] Если ассистент в чём-то затруднялся — обсудили
- [ ] Если ассистент сделал что-то очень хорошо — похвалил

### Уборка
- [ ] Помещение приведено в порядок (вместе с ассистентом)
- [ ] Оборудование выключено
- [ ] Если есть мусор / забытые вещи детей — собрано

### Если есть вопросы родителей
- [ ] Если ассистент / администратор передал вопрос родителя — ответил через фермера в течение 48 часов
- [ ] Не общается с родителями напрямую (только через фермера)

## ЧЕК-ЛИСТ 4. Ежедневный (если был день с уроками)

**Дата:** _____ **Тьютор:** _____ **Уроков сегодня:** ___

- [ ] Все уроки начались вовремя
- [ ] Чек-лист 2 (качество урока) самооценка по каждому
- [ ] Дневник тьютора заполнен по всем урокам
- [ ] Координация с ассистентом перед каждым уроком + после
- [ ] Если были эскалации (ребёнок / родитель / помещение) — переданы РФ или методисту

## ЧЕК-ЛИСТ 5. Недельный (заполняет в пятницу или субботу)

**Неделя с _____ по _____** **Тьютор:** _____

### Уроки
- [ ] Все запланированные уроки проведены (или есть уважительная причина пропуска)
- [ ] Дневники тьютора по всем урокам заполнены
- [ ] Чек-лист самооценки урока заполнен

### Развитие
- [ ] Минимум 1 урок другого тьютора просмотрен
- [ ] Методическая встреча сети (онлайн) посещена
- [ ] Если новые материалы от методиста — изучены

### Команда
- [ ] Если есть проблема с ассистентом — обсудил с РФ
- [ ] Если есть жалоба от родителя — передал методисту
- [ ] Если есть запрос от фермера — ответил

### Самооценка
- [ ] Какие 2-3 момента улучшить на следующей неделе

## ЧЕК-ЛИСТ 6. Ежемесячный (последняя неделя месяца)

**Месяц _____** **Тьютор:** _____

### Метрики (см. `05-pokazateli-i-istochniki.md`)
- [ ] Балл уроков (среднее по моим прослушанным методистом): ___ /10 (норма ≥ 7.5)
- [ ] Посещаемость в моих группах: ___% (норма ≥ 80%)
- [ ] Доля детей, освоивших тему месяца: ___% (норма ≥ 80%)
- [ ] Количество жалоб от родителей: ___ (норма 0)
- [ ] Количество отписок в моих группах: ___ (анализ причин)

### Отчёты прогресса родителям
- [ ] Заполнены на всех учеников
- [ ] Качество отчётов (не «формальная отписка»)
- [ ] Переданы фермеру

### Развитие
- [ ] Сколько чужих уроков посмотрел: ___
- [ ] Какие методические материалы изучил: ___

### Группы в зоне риска
- [ ] Если в одной из групп посещаемость < 70% — подготовлено объяснение и план для методиста / РФ
- [ ] Если есть «провалы» по освоению темы — план дополнительной работы

### Самооценка
- [ ] Что в этом месяце получалось лучше всего
- [ ] Что не получалось
- [ ] Что планирую улучшить в следующем месяце

## ЧЕК-ЛИСТ 7. Заполнение отчёта прогресса родителю (раз в месяц на каждого ученика)

См. шаблон: [`../../02-otchet-progressa-roditelyu.md`](../../02-otchet-progressa-roditelyu.md).

**Ребёнок:** _____ **Дата:** _____ **Месяц:** _____

### Что включается в отчёт
- [ ] Конкретные навыки, освоенные за месяц (минимум 2-3)
- [ ] Проекты, которые ребёнок сделал
- [ ] Что улучшилось vs прошлого месяца
- [ ] Что планируется на следующий месяц
- [ ] Личное наблюдение тьютора (отношение, мотивация)
- [ ] Рекомендации родителю (как поддерживать дома)

### Качество отчёта
- [ ] Не «шаблонная отписка» (текст индивидуальный)
- [ ] Конкретные примеры (не общие фразы)
- [ ] Соответствует фактической посещаемости (если ребёнок пропустил полмесяца — не пишем «активно работал»)
- [ ] Понятен родителю без технических терминов

### Передача
- [ ] Передан фермеру (через бот или вручную)
- [ ] Дата фиксирована в дневнике тьютора

## ЧЕК-ЛИСТ 8. Самостоятельная подготовка к аттестации (раз в полугодие)

**Дата аттестации:** _____ **Тьютор:** _____

### За 2 недели до
- [ ] Просмотрел свои метрики за 6 месяцев
- [ ] Прослушал 3-5 своих уроков (если есть запись) — самоанализ
- [ ] Подготовил список 3-5 ключевых достижений
- [ ] Подготовил список 2-3 зон роста (где осознанно знаю, что слабее)
- [ ] Подготовил вопросы / запросы к методисту и РФ

### Открытый урок (часть аттестации)
- [ ] Подготовил особенно тщательно (но без «театральных эффектов» — нужен реалистичный урок)
- [ ] Группа знает, что будут наблюдатели
- [ ] Готов к разбору после урока

### После аттестации
- [ ] Получил конкретный фидбэк (записал ключевое)
- [ ] Согласовал план развития на следующие 6 месяцев
- [ ] Если жёлтая зона — план развития на 3 месяца
- [ ] Если зелёная — обсудил карьерный рост (наставничество, мастер-классы)

## ЧЕК-ЛИСТ 9. Просмотр чужого урока (минимум 1 в неделю)

**Дата:** _____ **Чей урок:** _____ **Группа:** _____

### Перед просмотром
- [ ] Цель: «что я хочу узнать» (например: как работать с группой 7-летних / как объяснять Python новичкам)
- [ ] Тетрадь или ноутбук готов для заметок

### Во время урока
- [ ] Что мне понравилось (минимум 2 пункта)
- [ ] Что я делаю по-другому
- [ ] Что я хотел бы попробовать

### После просмотра
- [ ] Короткое спасибо коллеге (если был онлайн / лично)
- [ ] Заметки для себя (применить на своём уроке)

## ЧЕК-ЛИСТ 10а. Подготовка к Кибер-ярмарке (за 30-7 дней до)

Полный регламент: [`../../24-reglament-kiber-yarmarki.md`](../../24-reglament-kiber-yarmarki.md).

**Дата ярмарки:** _____ **Тьютор:** _____ **Группы:** _____

### День -30 (старт подготовки)
- [ ] Проведена сверка балансов киберонов всех своих детей в таблице [`24a-balans-kiberonov-template.csv`](../../24a-balans-kiberonov-template.csv)
- [ ] Все начисления и снятия за полугодие внесены (нет «по памяти»)
- [ ] Подготовлен список «зоны риска» (дети с балансом 0-30 киберонов)
- [ ] Дата ярмарки и подготовительный план получен от РФ

### День -21 (балансы родителям)
- [ ] Сформирована выгрузка баланса по каждому ребёнку
- [ ] Передана фермеру для отправки родителю (по шаблону из регламента)
- [ ] Если у родителя возникнут вопросы — готов показать историю начислений (журнал, а не «на словах»)

### День -14 (работа с «зоной риска»)
- [ ] С каждым ребёнком из «зоны риска» проведён разговор: «у тебя [N] киберонов, на ярмарке хватит на [...] — главное, что мы тебя ждём, кибероны копятся дальше»
- [ ] Родителям детей «зоны риска» — отдельное сообщение через фермера: «ребёнок может прийти как наблюдатель, чтобы не чувствовать себя в стороне»

### День -7 (встреча подготовки + финальный список)
- [ ] Посетил подготовительную встречу с РФ и тьюторами филиала
- [ ] Получил **финальный список баланса** своих детей (печатный + цифровой)
- [ ] Понятна моя роль в день ярмарки (кассир / поддержка младших / сопровождающий)
- [ ] Понятен ассортимент и цены товаров (из таблицы [`24b-zakupka-tovarov-template.csv`](../../24b-zakupka-tovarov-template.csv))

### День -1
- [ ] Финальный список балансов распечатан и есть с собой
- [ ] Если я кассир — ноутбук/планшет с таблицей [`24c-prodazhi-yarmarki-template.csv`](../../24c-prodazhi-yarmarki-template.csv) готов
- [ ] Если есть дети с очень низким балансом, которые могут расстроиться — план поддержки на день готов

## ЧЕК-ЛИСТ 10б. День Кибер-ярмарки (проведение)

**Дата:** _____ **Тьютор:** _____ **Моя роль:** _____ (кассир / поддержка / сопровождающий)

### Прибытие (за 30-45 минут до старта)
- [ ] Пришёл в офис вовремя
- [ ] Финальные списки балансов с собой
- [ ] Получил инструктаж от модератора (старшего тьютора / методиста)

### Если я «кассир» (учёт списания)
- [ ] **Каждая покупка** оформляется в [`24c-prodazhi-yarmarki-template.csv`](../../24c-prodazhi-yarmarki-template.csv) **сразу**
- [ ] Все обязательные поля заполнены: дата/время, ФИО, группа, товар, цена, баланс до/после, моё имя
- [ ] Перед списанием — проговариваю ребёнку: «[Товар] стоит [N] киберонов, баланс будет [M]. Уверен?»
- [ ] После подтверждения — товар выдаётся только через «продавца» прилавка
- [ ] Совместные покупки (когда дети скидываются) — оформляются двумя строками с пометкой
- [ ] Спорные ситуации / скидки — эскалирую модератору, не решаю сам
- [ ] Не было «оформлю потом» — все покупки в реальном времени

### Если я «поддержка младших» (для 6-8-летних)
- [ ] Каждый младший ребёнок получил мою помощь с подсчётом (если просил)
- [ ] Никто из младших не остался один у прилавка с растерянным видом
- [ ] Дети «зоны риска» прошли через ярмарку без слёз (с моей или родительской поддержкой)

### Если я «сопровождающий своей группы»
- [ ] Хожу рядом со своими детьми, помогаю с выбором
- [ ] Если сам не кассир — НЕ оформляю покупку «в обход», направляю к кассиру
- [ ] Слежу, чтобы мои дети успели купить (не «застряли в углу», не были оттеснены толпой)

### Общее
- [ ] Дисциплина (без криков) поддержана
- [ ] Помог фотографу/летописцу с эмоциональными кадрами
- [ ] Финальный «парад покупок» — мои дети показали, что купили

### Конец ярмарки
- [ ] Все покупки моих детей зафиксированы (проверка по списку)
- [ ] Никто из моих детей не «исчез» без покупки и без объяснения

## ЧЕК-ЛИСТ 10в. День +1 после Кибер-ярмарки (закрытие)

**Дата:** _____ **Тьютор:** _____

### Сверка
- [ ] Сумма списаний из [`24c-prodazhi-yarmarki-template.csv`](../../24c-prodazhi-yarmarki-template.csv) по моим детям = уменьшение балансов в [`24a-balans-kiberonov-template.csv`](../../24a-balans-kiberonov-template.csv)
- [ ] Расхождения отсутствуют ИЛИ переданы РФ + методисту для разбора
- [ ] Остаток киберонов каждого ребёнка зафиксирован (поле «остаток после ярмарки»)
- [ ] Перенос на следующее полугодие посчитан по правилу 30% от накопленного (см. регламент)

### Отчёт родителям
- [ ] На каждого ребёнка подготовлен индивидуальный отчёт по шаблону регламента:
  - что купил
  - сколько потратил
  - остаток
- [ ] Отчёты переданы фермеру для отправки родителям
- [ ] Если у ребёнка ярмарка прошла плохо (расстроен / не успел / поссорился) — отдельная заметка фермеру для звонка родителю

### Заметки для следующей ярмарки
- [ ] Какие 2-3 товара взяли «на ура» в моей группе — записал для РФ
- [ ] Какие 1-2 товара проигнорировали — записал, почему (на мой взгляд)
- [ ] Какие дети больше всего «копили» — почему, что их мотивирует, передал методисту
- [ ] Дети «зоны риска» — что нужно поправить в шкале начислений / в моей работе с ними?

### Самооценка
- [ ] Что у меня получилось на ярмарке
- [ ] Что я бы изменил
- [ ] Готов ли я на следующую быть на другой роли (например, перейти из «сопровождения» в «кассиры»)?

## КАК ХРАНЯТСЯ ЧЕК-ЛИСТЫ

**Где:**
- Google Drive: `KIBERone Management / Чек-листы тьюторов / [тьютор] / [год] / [месяц] /`
- Дневник тьютора — отдельная Google Form (наполняется ежедневно)
- Чек-листы 1, 2, 3 — простая таблица, заполняется быстро

**Доступ:**
- Тьютор — пишет
- Методист — читает (для оценки качества)
- РФ — читает (для оценки локальной работы)
- HR — читает аттестации

**Срок хранения:**
- Дневник тьютора — 12 месяцев
- Чек-листы качества — 12 месяцев
- Аттестации — бессрочно
- Отчёты прогресса родителям — 24 месяца
- Чек-листы Кибер-ярмарки (10а/10б/10в) — 24 месяца (для сверки между полугодиями)
- Таблицы балансов киберонов и продаж ярмарки — бессрочно (финансовый учёт)

**Правило:** если чек-лист не заполнен или потерян, считается, что соответствующая работа НЕ была сделана. Без дневника тьютора — премия за качество не выплачивается.
