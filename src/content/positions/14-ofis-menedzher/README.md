# Офис-менеджер (администратор филиала) — рабочая папка должности

**Кто такой офис-менеджер:** локальный администратор филиала. Первое лицо, которое видит родитель в офисе. Координатор расписания, документооборота, помещения, приёма родителей и детей.

**Зона ответственности:** один филиал — приёмная зона, расписание, помещение, документы, координация с тьюторами/ассистентами/фермерами.

**Главный продукт офис-менеджера:** **филиал, в котором всё работает гладко** — родители приходят и чувствуют себя комфортно, расписание не сбивается, оборудование не подводит, документы не теряются.

---

## Текущая ситуация (на 2026-05-01)

**Из оргструктуры:**
- В бэк-офисе сети есть «Офис-менеджер» — но это центральная роль, не локальная
- В каждом из 7 филиалов также нужен **локальный администратор** (фактически офис-менеджер на местах)
- В малых филиалах (Елабуга, Нижнекамск) роль может быть совмещена с другими (помощник менеджера развития, частичная занятость)
- В больших филиалах (Челны, Казань, Краснодар) — отдельная полноценная позиция

**Эта папка описывает офис-менеджера филиала** (локальная роль). Если в сети есть отдельный «офис-менеджер штаба» — для него отдельный документ при необходимости (но 80% содержания этой папки применимо).

**Подчиняется:**
- **Локально:** руководителю филиала (РФ)
- **Если РФ нет в этом городе:** напрямую собственнику + методически центральному офис-менеджеру (если есть)

**В подчинении:** никого (исполнительская роль).

## Главные ошибки в роли офис-менеджера (которые этой папкой пытаемся избежать)

1. **«Просто сидит на ресепшен»** — отвечает на телефон, но не координирует команду. Расписание сбивается, тьюторы недовольны.

2. **«Не первое лицо»** — при родителе грубовата / отвлечена / в телефоне. Родители получают первое впечатление о школе через офис-менеджера.

3. **«Не дисциплинирована в документах»** — теряет договоры, забывает заполнять журналы. Через 6 месяцев — бухгалтерский кошмар.

4. **«Не контролирует помещение»** — оборудование сломано, расходники кончились, мусор не выносится. Тьюторы тратят своё время на эти проблемы.

5. **«Не отвечает на WhatsApp / телефон»** — родители звонят в офис, никто не берёт трубку. Часть теряется, часть идёт к фермеру / РФ, перегружая их.

Все 5 ошибок — описаны с тем, как их избегать.

## Особенности роли в KIBERone-сети

1. **Офис-менеджер — главная точка контакта родителя с филиалом.** Если фермер общается с родителем удалённо, то офис-менеджер видит родителей лично, когда они приводят / забирают детей. Это уникальный контакт.

2. **Сезонность:** сентябрь и январь — пиковые (новые родители, новые группы, много документов). Лето — спокойнее, но есть лагерь (своя специфика).

3. **Много рутинных задач, мало стратегических.** Это исполнительская роль. Главное — стабильность и качество базовых процессов.

4. **Связь с продажами и удержанием.** Если офис-менеджер «холодная» — родитель чувствует это и думает «может, и обучение такое же холодное?». Прямая связь с оттоком.

## Норма нагрузки

| Параметр | Норма (full-time) |
|---|---|
| Часов в смене | 6-8 (зависит от расписания филиала) |
| Дней в неделю | 5-6 |
| Скорость ответа на звонок | 3-5 гудков |
| Скорость ответа на WhatsApp | ≤ 30 минут в рабочее время |
| Время приёма родителя на ресепшен | ≤ 5 минут (ожидание родителя) |
| Документы оформлены без ошибок | 100% (договоры, платежи, чеки) |
| Помещение готово к занятиям | 100% (за 30 минут до первого урока) |
