# Методист — рабочая папка должности

**Кто такой методист:** центральный сотрудник сети (не привязан к одному филиалу). Отвечает за качество учебного процесса по всей сети — программа, тьюторы, ассистенты, аттестации, контроль уроков. Подчиняется собственнику (или операционному директору, если он реально берёт функцию).

**Зона ответственности:** 7 филиалов, ~30 тьюторов, ~40 ассистентов, ~1 640 учеников.

**Главный продукт методиста:** стабильное качество обучения во всех филиалах — измеряется через прогресс детей, удержание тьюторов, удовлетворённость родителей.

## Как пользоваться этой папкой

**Если нанимаете нового методиста:**
1. Дать прочитать `01-rol-i-funkcii.md` ДО собеседования
2. После найма — запустить программу `02-sistema-adaptacii.md`
3. Через месяц — оценить по `07-kpi-i-mesyachnye-metriki.md`

**Если методист уже работает:**
1. Сверить факт с `01-rol-i-funkcii.md` — что делает по факту, что должен делать
2. Проверить `04-cheklisty.md` — выполняет ли регулярные процедуры
3. Запустить замер по `05-pokazateli-i-istochniki.md` за последние 1-3 месяца
4. По результатам — корректировка зоны ответственности или увольнение

**Если методиста нет (роль не выделена):**
- Сейчас часть функций методиста выполняет операционный директор — но не системно
- Документы 01 и 02 — основа для найма
- Срочность: средняя. Без методиста сеть жива, но качество дрейфует, тьюторы выгорают, текучка растёт.
