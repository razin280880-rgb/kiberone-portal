# РОР: KPI и ежемесячные метрики

KPI РОР, ежемесячный дашборд для собственника.

---

## KPI РОР — 5 ключевых метрик

### KPI 1. Поток квалифицированных анкет (главный KPI)

**Метрика:** количество анкет в статусе «записан на пробное» за месяц.

**Вес в оценке:** 40%.

**Целевое значение:** план на месяц (задаётся квартально, исходя из плана по выручке нового бизнеса).

**Шкала:**
| Факт vs план | Зона |
|---|---|
| ≥ 110% | Выше нормы |
| 100–110% | Норма |
| 90–99% | Лёгкое отставание |
| 80–89% | Жёлтая |
| 70–79% | Красная |
| < 70% | Красная, разбор |

### KPI 2. Качество анкет

**Метрика:** доля «грязных» анкет, переданных в РОП за месяц.

**Вес в оценке:** 20%.

**Целевое значение:** ≤ 10%.

**Шкала:**
| Доля грязных | Зона |
|---|---|
| ≤ 5% | Выше нормы |
| 5–10% | Норма |
| 10–15% | Лёгкое отставание |
| 15–20% | Жёлтая |
| 20–25% | Красная |
| > 25% | Красная, разбор |

### KPI 3. Доходимость до пробного

**Метрика:** доля записавшихся, которые реально пришли на пробное за месяц.

**Вес в оценке:** 15%.

**Целевое значение:** ≥ 70%.

**Шкала:**
| Доходимость | Зона |
|---|---|
| ≥ 80% | Выше нормы |
| 70–80% | Норма |
| 60–70% | Жёлтая |
| 50–60% | Красная |
| < 50% | Красная, разбор |

### KPI 4. Диверсификация каналов

**Метрика:** **ни один канал не превышает 50% от всех анкет месяца** + минимум 4 активных канала на каждый город.

**Вес в оценке:** 10%.

**Шкала (двоичная):**
- Выполнено по обоим условиям — норма
- Нарушено одно условие — жёлтая
- Нарушены оба — красная

### KPI 5. Управленческая ритмика

**Метрика:** комплексная — проведение планёрок, 1-1, обучений по нормам.

**Вес в оценке:** 15%.

**Подкомпоненты:**
- Ежедневная планёрка: ≥ 95% рабочих дней
- Стратегическая планёрка: 100% посещение
- 1-1 с каждым менеджером развития: раз в 2 недели
- Еженедельная обучающая сессия: ≥ 90% запланированных в квартале
- Прослушивание звонков: ≥ 25 в неделю
- Аттестация: 100% сотрудников отдела имеют аттестацию ≤ 6 месяцев

**Шкала:**
| Сколько подкомпонентов выполнено | Зона |
|---|---|
| 6/6 | Выше нормы |
| 5/6 | Норма |
| 4/6 | Жёлтая |
| 3/6 | Красная |
| < 3/6 | Красная, разбор |

## Квартальная оценка

Дополнительная оценка раз в квартал при условии:

- Все 3 месяца квартала — выручка нового бизнеса ≥ 95% плана
- Все 3 месяца — взвешенная доля KPI ≥ 85%
- Нет действующих письменных предупреждений по дисциплине

---

## Годовая оценка

Дополнительная оценка раз в год при условии:

- Все 4 квартала — KPI выполнен ≥ 90%
- Выполнен годовой план по выручке нового бизнеса ≥ 95%
- Минимум 4 запущенных и масштабированных новых канала за год
- Доля «зелёных» менеджеров развития к концу года ≥ 70%
- Минимум 8 новых школьных партнёрств в сети за год

## Ежемесячный отчёт для собственника / ОД

РОР сдаёт **до 3-го рабочего дня нового месяца** отчёт по шаблону:

### Раздел 1. Цифры (объёмные)

- Анкеты получено: всего / по городам / по каналам
- Записи на пробное: всего / план / факт
- Доходимость до пробного
- Конверсия квалификации
- Активные промо-точки и партнёрства

### Раздел 2. Качество

- Доля грязных анкет
- Заполняемость обязательных полей
- Средний балл звонков менеджеров

### Раздел 3. Финансы

- CAC по каналам
- Распределение бюджета по каналам
- Изменения vs прошлого месяца

### Раздел 4. Команда

- Состояние менеджеров развития (зоны)
- Состояние промоутеров
- Найм/увольнения

### Раздел 5. Выводы и решения

- 3 главные удачи месяца
- 3 главные провала месяца
- 3 решения на следующий месяц

### Раздел 6. Запросы к собственнику / ОД

- Что нужно от собственника
- Какие решения требуют согласования
- Какие ресурсы нужны (бюджет, найм, ИТ-инструменты)

## Связь KPI РОР с KPI других руководителей

| KPI | Связь с другими |
|---|---|
| Поток анкет | Прямо влияет на план РОП (конверсия → продажа) |
| Качество анкет | РОП теряет время на «грязных» — снижается его конверсия |
| Доходимость | Совместная зона с РОП (звонок-напоминание хантера) |
| Диверсификация каналов | Снижает риск для всей сети |
| Ритмика | Влияет на качество команды → косвенно на все остальные KPI |

**Кросс-KPI с РОП:** общая «конверсия анкета → договор». РОР отвечает за качество анкеты, РОП — за конверсию. Если общая конверсия падает — совместный разбор и совместная корректировка.
