# Чек-листы офис-менеджера

Все чек-листы — рабочий инструмент. Заполнение фиксируется (галочка с датой), хранится минимум 6 месяцев.

**Логика:** офис-менеджер — это **роль рутины и стабильности**. Чек-листы — основа всей работы. Без них процессы рушатся за неделю.

---

## ЧЕК-ЛИСТ 1. Открытие смены (за 30 минут до первой группы)

**Дата:** _____ **Офис-менеджер:** _____

### Помещение
- [ ] Офис открыт, освещение включено
- [ ] Приёмная зона чистая, мебель на местах
- [ ] Кулер заправлен (вода, стаканчики)
- [ ] Гардероб готов (вешалки, обувница)
- [ ] Туалеты чистые, есть бумага и мыло
- [ ] Все кабинеты проверены: чистые, проветрены, оборудование на месте

### Оборудование
- [ ] Все компьютеры в кабинетах включены и проверены
- [ ] Проекторы работают (если используются сегодня)
- [ ] Интернет работает (тест)
- [ ] Принтер заправлен бумагой
- [ ] Касса включена, остаток сверен с вчерашним отчётом

### Расходные материалы (если предполагается использование сегодня)
- [ ] Маркеры на досках
- [ ] Бумага в принтере
- [ ] Канцелярия в кабинетах
- [ ] Раздаточные материалы для уроков (если есть)

### Документы и информация
- [ ] CRM открыт, расписание дня просмотрено
- [ ] Список ожидаемых детей готов
- [ ] Если есть пробное — карточка клиента готова
- [ ] Чаты прочитаны, нет срочных вопросов от РФ / методиста

### Команда
- [ ] Тьюторы / ассистенты на местах (или скоро придут)
- [ ] Если кто-то опаздывает — есть понимание почему

**Если хотя бы одна галочка не стоит** — фиксировать в чате РФ.

---

## ЧЕК-ЛИСТ 2. Закрытие смены (после последней группы)

**Дата:** _____ **Офис-менеджер:** _____

### Дети
- [ ] Все дети забраны родителями
- [ ] Если кто-то задерживается — связались с родителем
- [ ] Никто не забыт в кабинете

### Кабинеты
- [ ] Все компьютеры выключены
- [ ] Свет выключен в кабинетах
- [ ] Двери кабинетов закрыты
- [ ] Никто не оставил вещей (если оставил — в «забытые вещи»)

### Помещение
- [ ] Приёмная зона убрана
- [ ] Мусор вынесен
- [ ] Кулер проверен (есть ли вода на завтра)

### Касса
- [ ] Дневной отчёт сделан
- [ ] Остаток сверен
- [ ] Если есть наличные → инкассация / сейф
- [ ] Чеки за день сложены, готовы к передаче в бухгалтерию

### Документы
- [ ] Все договоры за день оформлены и подшиты
- [ ] Все журналы заполнены
- [ ] Если есть проблемы / жалобы — отправлены в чат РФ

### Информация
- [ ] CRM проверен: все статусы дня обновлены
- [ ] План на завтра просмотрен
- [ ] Если есть что-то срочное на завтра — записано

### Закрытие
- [ ] Окна закрыты
- [ ] Электроприборы выключены (кроме оставленных нарочно)
- [ ] Сигнализация включена (если есть)
- [ ] Дверь закрыта на замок

## ЧЕК-ЛИСТ 3. Приём родителя (новый клиент пришёл в офис)

**Родитель:** _______ **Дата:** _______

### Встреча
- [ ] Поздороваться (встать, улыбнуться)
- [ ] Спросить, чем могу помочь
- [ ] Если первый раз — представиться

### Если интересуются обучением
- [ ] Узнать имя ребёнка, возраст
- [ ] Рассказать общую информацию (1-2 минуты, не больше)
- [ ] Предложить пробное занятие
- [ ] Узнать контакт (телефон, WhatsApp) — занесла в анкету в CRM
- [ ] Передать хантеру (или менеджеру развития) — звонок в течение 24 часов

### Если родитель уже клиент
- [ ] Узнать имя ребёнка / на какую группу пришёл
- [ ] Проверить расписание
- [ ] Проводить в нужный кабинет / зону ожидания

### Если есть вопрос / претензия
- [ ] Внимательно выслушать (не перебивать)
- [ ] Не спорить, не оправдываться
- [ ] Если простой вопрос — ответить
- [ ] Если сложный — «давайте я свяжу вас с [методист / фермер / РФ]»
- [ ] Зафиксировать обращение в CRM

### После
- [ ] Внесена анкета (если новый клиент)
- [ ] Передан нужный сотрудник
- [ ] Если что-то обещала родителю — записала в CRM с дедлайном

---

## ЧЕК-ЛИСТ 4. Ответ на звонок офиса

### При звонке
- [ ] Ответить в 3-5 гудков (быстро, не «потом»)
- [ ] Стандартное приветствие: «Школа KIBERone, [имя], добрый день»
- [ ] Слушать вопрос полностью, не перебивать

### Идентификация запроса
- [ ] Простой вопрос (расписание, цены, общая информация) — отвечаю сама
- [ ] Сложный вопрос (продажи, методический, удержание) — переадресую:
  - Новый клиент / продажа → хантер / менеджер развития
  - Учебный процесс / жалоба → методист
  - Удержание / отписка / лагерь → фермер
  - Общие проблемы филиала → РФ

### Передача
- [ ] Если не отвечает «нужный человек» — записать звонок (имя + телефон + суть)
- [ ] Передать в чат соответствующему сотруднику
- [ ] Контролировать: позвонил ли он в течение часа

### После
- [ ] Зафиксировать в CRM (если новый клиент или важный звонок)
- [ ] Если есть «обещание перезвонить» — поставить напоминание

## ЧЕК-ЛИСТ 5. Оформление договора (новый клиент)

**Клиент:** _______ **Дата:** _______ **Хантер:** _______

### Подготовка
- [ ] Шаблон договора готов (актуальная версия)
- [ ] Известны условия от хантера (тариф, скидка если есть)

### Оформление
- [ ] Все поля заполнены без ошибок:
  - ФИО родителя
  - Контакты
  - ФИО ребёнка, дата рождения
  - Группа / тариф / срок
  - Сумма
- [ ] Подпись родителя получена
- [ ] Копия договора выдана родителю
- [ ] Оригинал — в подшивку филиала

### Платёж
- [ ] Сумма принята (наличные / карта)
- [ ] Чек выдан
- [ ] Транзакция в CRM

### CRM
- [ ] Карточка клиента создана
- [ ] Тег «новый клиент»
- [ ] Передан фермеру (через CRM или чат)

### Контроль
- [ ] Договор на месте
- [ ] Хантер уведомлён о завершении оформления
- [ ] Если есть особые условия — записаны в карточке

## ЧЕК-ЛИСТ 6. Перенос занятия (по запросу родителя или тьютора)

**Запрос от:** родитель / тьютор / фермер **Дата:** _______ **Старая дата:** _______ **Новая дата:** _______

### Согласование
- [ ] С тьютором, который ведёт группу — согласовано
- [ ] С другими родителями (если групповое занятие) — информированы
- [ ] Свободность кабинета и оборудования проверена

### Внесение
- [ ] CRM обновлён (старая дата — отметка, новая — записана)
- [ ] Расписание обновлено
- [ ] Тьютор и ассистент в курсе
- [ ] Родитель получил подтверждение

### Если перенос на отработку (после пропуска)
- [ ] Согласовано с фермером (это его процесс)
- [ ] В CRM фиксируется как «отработка пропуска»

**Период:** _______ **Дата передачи:** _______

- [ ] Все договоры за период (оригиналы)
- [ ] Все чеки за период (или выгрузка из кассы)
- [ ] Журнал кассовых операций
- [ ] Журнал материальных средств (если есть)

## ЧЕК-ЛИСТ 8. Подготовка к мероприятию (День открытых дверей и т.п.)

**Мероприятие:** _______ **Дата:** _______

### За 7 дней до
- [ ] Получено ТЗ от менеджера развития: программа, время, гости
- [ ] Согласовано: какие кабинеты задействованы, кто из тьюторов проводит

### За 3 дня до
- [ ] Запас расходных материалов (бумага, маркеры, бэйджи, угощение)
- [ ] Подготовлены анкеты для гостей

### В день мероприятия
- [ ] Помещение убрано особенно тщательно
- [ ] Регистрационный стол готов
- [ ] Команда на местах
- [ ] Оборудование проверено

### Во время
- [ ] Регистрация гостей
- [ ] Выдача бэйджей / анкет
- [ ] Координация по программе

### После
- [ ] Анкеты собраны → передачу менеджеру развития
- [ ] Помещение убрано
- [ ] Отчёт о мероприятии (количество гостей, особенности)

## КАК ХРАНЯТСЯ ЧЕК-ЛИСТЫ

**Где:**
- Google Drive: `KIBERone Management / Чек-листы офис-менеджеров / [филиал] / [год] / [месяц] /`
- Чек-листы открытия / закрытия — заполняются ежедневно (в коротком формате)
- Ежедневный / ежемесячный — отдельные файлы

**Доступ:**
- Офис-менеджер — пишет
- РФ — читает
- HR — читает (для аттестации)

**Срок хранения:**
- Ежедневные / еженедельные — 6 месяцев
- Ежемесячные — 12 месяцев
- Аттестации — бессрочно
- Документы (договоры) — по бухгалтерскому регламенту (5 лет)

**Правило:** если чек-лист открытия / закрытия не заполнен — премия за стабильность не выплачивается.
